## 一、计算题必拿30分（练2遍即可满分）

### ① 均值滤波 & 中值滤波（10分）

**3×3 均值滤波器**：9个 1/9

**3×3 中值滤波器**：取窗口内9个像素排序后的中值（第5个数）

**计算步骤**：
- 5×5 图用 3×3 核做 valid 卷积，输出 3×3
- 窗口中心在原图第2~4行、第2~4列
- **均值**：每个窗口9个数求和÷9
- **中值**：每个窗口9个数排序取第5个

**特点对比**：

| 滤波器 | 原理 | 对哪种噪声好 | 原因 |
|---------|------|-------------|------|
| 均值滤波 | 窗口求平均 | **高斯噪声** | 噪声均值=0，平均后相互抵消 |
| 中值滤波 | 窗口取中位数 | **椒盐噪声** | 椒盐是极端值，排序后被挤到两端，中值不受影响 |

> 🔤 **口诀：均值治高斯，中值治椒盐**

### ② 直方图均衡化（10分）

**4步公式**：
1. 统计概率 p(k)
2. 求累积 CDF：s(k) = p(0)+p(1)+…+p(k)
3. 映射：round( s(k) × (L−1) )，L为灰度级数
4. 写映射表，合并同值概率

**真题示范**（8灰度级 0~7）：
- p = [0.174, 0.088, 0.086, 0.080, 0.068, 0.058, 0.062, 0.384]
- **答案：0→1, 1→2, 2→2, 3→3, 4→3, 5→4, 6→4, 7→7**

> 🔤 **口诀：累加 → 乘7 → 四舍五入**

⚠️ **易错**：8级乘7，16级乘15，通用 L−1

### ③ 边缘检测算子（10分）

**必背模板**：

**Sobel Gx**（3×3，中心加权2倍，抗噪最好）：
```
-1  0  1
-2  0  2
-1  0  1
```

**Sobel Gy**：
```
-1 -2 -1
 0  0  0
 1  2  1
```

**Prewitt Gx**（无权值）：
```
-1  0  1
-1  0  1
-1  0  1
```

**Laplacian**（二阶，4邻域）：
```
 0  1  0
 1 -4  1
 0  1  0
```

**Roberts Gx**（2×2，对角）：
```
 1  0
 0 -1
```

**计算步骤**：
1. 边界填0（原图5×5 → 7×7）
2. 算子翻转180°后与窗口对应相乘再求和（Sobel对称，翻转不变）
3. 梯度幅值 = |Gx| + |Gy|

## 二、选择题速记（40分·20题）

### 核心公式速查

| 考点 | 结论 |
|------|------|
| 存储比特数 | 宽×高×每像素bit数；N级灰度需 log₂N bit |
| valid卷积输出 | (H−k+1) × (W−k+1)；乘法次数 = 输出像素数 × k² |
| 可分离卷积 | 2D核可分离时，O(k²) → O(2k) |
| 高斯核大小 | ≈ 2×(3σ)+1（覆盖±3σ） |
| 平滑 vs 锐化 | 平滑=低通(去噪)；锐化=高通(突出边缘) |
| 腐蚀 vs 膨胀 | 腐蚀=缩小(取min)；膨胀=放大(取max) |
| PCA vs LDA | PCA无监督/最大化方差；LDA有监督/最大化类间分离 |

### 真题答案速记

| 题号 | 答案 | 关键点 |
|------|------|--------|
| 1 | B(196Kb) | 256×256×3=196608bit |
| 2 | A | BRDF与角度、材料都有关 |
| 3 | A(HSV) | HSV/HSI是非线性颜色空间 |
| 4 | C(对比度) | HSV含H/S/V，不含对比度 |
| 5 | C错 | a=1,b>0变亮(不是变暗) |
| 6 | B(光线明暗) | 几何关系与亮度无关 |
| 7 | D(均方误差) | 均方误差是回归指标 |
| 8 | D | CNN需标注数据、核是学习的 |
| 9 | A(1764) | (10-7+1)×(15-7+1)×49=1764 |
| 10 | D(高通) | 锐化=高通滤波 |
| 11 | D | 腐蚀→变小/收缩 |
| 12 | B(k-NN) | k-NN是分类器不是聚类 |
| 13 | C错 | 二维FT具有可分离性 |
| 14 | C | 低通=高频受抑制，低频通过 |
| 15 | D(Laplacian) | Laplacian是二阶算子 |
| 16 | C(Sobel) | Sobel中心加权抗噪最好 |
| 17 | B | 对角线模板检测对角边缘 |
| 18 | B | Canny四步无傅里叶变换 |
| 19 | D错 | 角点是曲率局部极大点(非极小) |
| 20 | D | PCA不利用类别信息，LDA才利用 |

### 一阶 vs 二阶算子对比

| 对比项 | 一阶(Roberts/Prewitt/Sobel) | 二阶(Laplacian/LOG) |
|--------|---------------------------|---------------------|
| 原理 | 求一阶导数(梯度) | 求二阶导数 |
| 边缘 | 较粗 | 过零点定位，较细 |
| 对噪声 | 较敏感 | **更敏感** |
| 抗噪最好 | **Sobel**(中心加权) | **LOG**(先高斯平滑) |

> ❗ **易错**：「二阶比一阶对噪声敏感度低」→ **错**，二阶更敏感

## 三、判断题速记（20分·10题）

| 题 | 陈述 | 答案 | 理由 |
|----|------|------|------|
| 1 | 分辨率是指图像所有像素组成的尺寸大小 | **对** | 空间分辨率=像素尺寸/数量 |
| 2 | 直方图均衡化对灰度分布均衡的图效果明显 | **错** | 对分布不均的图效果才明显 |
| 3 | 均值滤波器可用于锐化边缘 | **错** | 均值是平滑，会模糊边缘 |
| 4 | SIFT对尺度变换没有不变性 | **错** | SIFT=Scale-Invariant |
| 5 | 小方差/小尺寸高斯核平滑能力强 | **错** | 核越小平滑越弱 |
| 6 | Hough变换可检测直线、圆、椭圆等 | **对** | 参数空间投票法 |
| 7 | 二阶微分算子比一阶对噪声敏感度低 | **错** | 二阶更敏感 |
| 8 | 空域卷积等价于频域相乘 | **对** | 卷积定理 |
| 9 | 图像分割可看作像素的聚类问题 | **对** | 相似像素聚成一类 |
| 10 | 斑点检测中半径是LoG标准差的常数倍 | **对** | r = σ√2 |

> 🔤 **口诀：对 错 错 错 错 / 对 错 对 对 对**（前5题只第1对，后5题只第7错）

## 四、综合题模板（10分）

### 汽车车牌自动识别6步流程

```
1. 图像采集 → 摄像头获取含车牌的彩色图像
2. 预处理 → ①灰度化 ②去噪(高斯/中值) ③对比度增强(直方图均衡化)
3. 车牌定位 → 边缘检测(Sobel/Canny) + 形态学(膨胀腐蚀) + 连通域分析
4. 字符分割 → 阈值二值化 → 倾斜校正 → 投影法/连通域切分
5. 字符识别 → 特征提取(PCA/HOG) + 分类器(k-NN/SVM/CNN)
6. 后处理 → 按编排规则校验 → 输出结果
```

## 五、考纲22点速查

### 成像与视觉基础（1~5）
- 透镜聚焦成像；光圈越大景深越浅
- 杆体=暗视觉(亮度无色)；锥体=明视觉(色彩3种)
- BRDF与角度、材料有关
- 成像三要素：光源+物体+相机
- CV趋势：深度学习、多模态、大模型

### 颜色空间与图像基础（6~9）
- RGB(加色)；CMY(减色)；XYZ(线性标准)；**HSI/HSV(非线性)**；YUV(视频)；Lab(感知均匀)
- N级灰度需 log₂N bit
- 点运算只依赖单个像素；局部运算依赖邻域
- 线性 g=af+b；非线性：对数(扩展暗部)、伽马(γ<1变亮)

### 直方图与形态学（10~11）
- 直方图均衡化：CDF变换，累加→×(L−1)→取整
- 膨胀=变大(取max)；腐蚀=变小(取min)

### 卷积与滤波（12~14）
- 卷积：反褶、交换律、可分离性、边界填充
- 高斯核≈2×3σ+1；可分离复杂度O(2k)
- 空域卷积=频域相乘；低通平滑/高通锐化

### 边缘与特征（15~17）
- Canny四步：高斯平滑→梯度→非极大值抑制→双阈值
- Hough变换：参数空间投票(ρ=xcosθ+ysinθ)
- 分割：阈值/边界/区域

### 聚类·关键点·检测·分类（18~22）
- k-means(需K·对初值敏感)；mean shift(无需K)
- Harris角点：R=det(M)−k·trace²(M)
- SIFT：尺度不变+旋转不变，128维描述子
- Adaboost+Haar+积分图+金字塔 → Viola-Jones人脸检测
- **PCA算法**：中心化→协方差→特征值分解→取前k大→投影

## 六、全部口诀汇总

1. **均值治高斯，中值治椒盐**

11. **PCA无监督最大化方差；LDA有监督利用类别信息**
