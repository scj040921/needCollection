# 1.整型
- **概念**：表达整数类型的数据
- **整型修饰符**：
    - short：短整型，用来缩短整型变量的尺寸
    - long：长整型，用来增长整型变量的尺寸
    - long long：长长整型，用来增长整型变量的尺寸
    - unsigned：用来去除整型变量的符号位，使得整型变量只能表达非负整数
- **格式控制符**：
	1. int整型：%d
	2. short整型：%hd
	3. long整型：%ld
	4. long long整型：%lld
- 使用整型修饰符后关键字int可以省略
- **符号位**：
	- 有符号的整型数据，首位为符号位，0表示正数，1表示负数
	- 无符号的整型数据，没有符号位
- **编码形式**：
	- 源码：正数直接使用二进制来表达
	- 补码：负数用绝对值取反加一的时候，符号位不动
