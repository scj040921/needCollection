# 一、头文件
1. C语言程序中，任何符号（包括变量和函数）的使用，都是需要提前声明的。
2. 如果程序使用到了系统库函数，那么就需要包含相应的系统头文件来进行声明。
3. 声明是指对源文件外定义的某个符号进行描述的过程，经过编译的符号，编译器才能正确的识别和编译。
# 二、主函数
1. 主函数是必须有的，每一个C语言程序都有main函数，哪怕是空的。
2. 主函数的名字是固定的，这个函数名不可挪作他用。
3. 主函数是程序的入口，也就是程序开始运行的地方。
# 三、输出函数printf()
- **概念**：函数printf被称为格式化IO函数，因为它是按照指定格式来操作数据的。
- **语法点**：
	- 类似%d、%f被称为格式控制符
	- 一个格式控制符对应一个数据
	- \n是换行符
# 四、函数返回return
C语言中的关键字return的含义是结束一个函数，并返回某个数据（若有），而具体到main函数中时，由于主函数的退出同时也意味着整个程序的结束，一般而言程序正常退出时，main函数的返回值是0；当程序异常退出时，main函数的返回值为非0.
**注意**：main函数的返回值是给上一级进程的，因此理论上在单进程程序中，main函数的返回值是多少无所谓
