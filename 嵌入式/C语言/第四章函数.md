# 1. 函数的基本概念
- 在C语言中，函数意味着功能模块。典型的C语言程序是由多个功能模块拼接组成，因此C语言被称为模块化语言。函数是封装的，对外提供的接口简洁，内部功能明确
1. **函数定义**
	- - 典型函数定义的语法如下：
		返回类型 函数名称(参数1, 参数2, ……)
		{
		    函数体
		}
	- 函数头：第一行称为函数头，或称函数特征表，是函数对外的接口
		1. 函数名称：命名规则与变量一致
		2. 参数列表：即黑箱的输入数据列表，一个函数可以有多个参数也可以没有参数
		3. 返回类型：即黑箱的输出数据类型，一个函数可以不返回数据，但最多只能返回一个数据。
	- 函数体：花括号内为函数体，是函数功能的内部实现
	- 语法：
		1. 当函数的参数列表为void时，表示该函数不需要任何参数
		2. 当函数的返回类型为void时，表示该函数不返回任何数据
		3. 关键字return表示退出函数。若函数头中规定有返回数据类型，则return需要携带一个类型与之匹配的数据；若函数头中规定返回类型为void，则return不需携带参数。
2. **函数调用**
	- 函数调用必须携带圆括号，里面依次放入函数定义中所规定的参数列表，参数的类型、个数、顺序都必须跟定义一致。
3. **函数声明**
	- 先声明再调用，顺序不可反
