# 1.结构体
## 基本概念 
- 结构体是c语言中用来将多种不同的数据类型组合起来，形成某种现实意义的自定义的变量类型，本质上是一种自定义类型
	```
	struct 结构体标签
	{
		成员1;
		成员2;
		...
	};
	```
- **语法**：
	- 结构体标签用来区分各个不同的结构体
	- 成员是包含在结构体内部的数据，可以是任意的数据类型
- **注意**：
	- 空结构体的大小不为0
	- 因为内存对齐机制，sizeof的结果不等于成员大小之和
## 结构体初始化
- **结构体的定义和初始化**：
	- 结构体内部拥有多个不同类型的成员，初始化采用与数组类似的列表方式
	- 初始化方式：
		1. 普通初始化
		2. 指定成员初始化
- **指定成员初始化的好处**：
	- 成员初始化的次序可以改变
	- 可以初始化一部分成员
	- 结构体新增成员之后初始化语句依然可用
## 结构体成员引用
结构体相当于一个集合，其中的每个成员都是独立的变量，都可以被独立的引用
## 结构体数组与结构体指针
- 结构体指针：
	- 
```
struct node  n = {100, 'x', 3.14};
struct node *p = &n;

// 以下语句都是等价的
printf("%d\n",   n.a);
printf("%d\n", (*p).a);
printf("%d\n",  p->a);  // 箭头 -> 是结构体指针的成员引用符
```
- 结构体数组：
	```
struct node s[5];
s[0].a = 300;
s[0].b = 'z';
s[0].c = 3.45;
```
