# Part 一：成像基础（考纲1~5）

---

## 1. 透镜与成像基本原理

### 通俗解释

想象你拿一个放大镜对着太阳，能把光聚到一个点上——这就是透镜成像的核心。透镜通过**折射**（光穿过玻璃会弯折）把物体上每一点发出的光汇聚到传感器的对应位置上，形成清晰的像。

**小孔成像**是更古老的方法：在暗箱上开个小孔，外面的光通过小孔在对面墙上形成倒立的像。原理是**光的直线传播**。小孔越小→像越清晰但越暗；小孔越大→像越亮但越模糊。

### 📌 背记要点

| 参数         | 含义                          | 记忆口诀                            |
| ---------- | --------------------------- | ------------------------------- |
| **焦距 f**   | 透镜光心到焦点的距离                  | 长焦=远/窄（拍远景），短焦=近/广（拍全景）         |
| **光圈（f值）** | 控制进光量的开口大小                  | f值越小 → 光圈越大 → 进光越多 → 景深越浅（背景虚化） |
| **薄透镜公式**  | `1/f = 1/u + 1/v`，u=物距，v=像距 | 物距无穷远时，像距=焦距                    |

✅ **景深记忆**：f值小→光圈大→景深浅（虚化）；f值大→光圈小→景深深（全清晰）

## 2. 人类视觉系统（人眼）

### 通俗解释

人眼就像一台**自动对焦的照相机**。光线进入眼睛后依次穿过：角膜（前挡风玻璃）→ 瞳孔（光圈）→ 晶状体（变焦镜头）→ 玻璃体 → 最后落在视网膜（感光底片）上。

视网膜上有两种感光细胞，这是**选择题高频考点**：

| 细胞类型 | 数量 | 功能 | 场景 |
|----------|------|------|------|
| **杆体细胞** | 约1.2亿个 | 只感明暗（灰度），不感颜色 | 夜间/暗处视觉 |
| **锥体细胞** | 约600万个 | 感知颜色，分L(红)、M(绿)、S(蓝)三种 | 白天/明亮处+色彩感知 |

### 📌 背记要点

🔤 **口诀**：杆体管夜视（感光强、无色），锥体管彩色（感光弱、有色）

| 人眼部件 | 功能                   | 对应相机        |
| ---- | -------------------- | ----------- |
| 角膜   | 折射光线（约占总折射力的**2/3**） | 镜头前组        |
| 瞳孔   | 控制进光量（暗处放大、亮处缩小）     | 光圈          |
| 晶状体  | 改变形状调节焦距（看近变厚、看远变薄）  | 对焦系统        |
| 视网膜  | 感光成像（含杆体+锥体细胞）       | CCD/CMOS传感器 |

## 3. 光度学、色度学与BRDF

### 通俗解释

**辐照度**（Irradiance）：物体表面**接收到**的光有多强（入射光），单位面积的辐射功率。
**辐射度**（Radiance）：物体表面**反射出去**的光有多强（出射光），向某个方向辐射的功率。

**BRDF**（双向反射分布函数）描述的就是：一束光从某个方向照到物体上，在另一个方向反射出多少光。

⚠️ **易错点**：BRDF与**入射角度、观察角度、表面材料**三个因素都有关——不是只与其中一个有关！

### 📌 背记要点

- **亮度**（Luminance）：人眼感知的明暗，对555nm黄绿光最敏感
- **色度**（Chromaticity）：颜色种类和纯度，由色度坐标(x,y)描述
- **BRDF公式**：`fr = dLr / dEi`（反射辐射度增量 / 入射辐照度增量）
- **成像三要素**：光源 + 物体 + 传感器（三者缺一不可）

## 4. 成像过程三要素

### 通俗解释

成像就像拍照：得有**光源**（灯/太阳）照亮物体，得有**物体**（你拍的东西），还得有**传感器**（相机底片/CCD）。少任何一个都拍不成。

### 📌 背记要点

✅ 成像三要素：**光源 + 物体 + 传感器**

成像过程链：光源发光 → 照射物体 → 物体反射光线 → 透镜聚焦 → 传感器接收 → 数字化为图像

## 5. 计算机视觉定义与发展趋势

### 📌 背记要点

- **定义**：用计算机和算法模拟人眼，从图像/视频中提取、分析和理解信息
- **核心任务递进**：图像分类（是什么？）→ 目标检测（在哪？）→ 语义分割（每个像素属于什么？）
- **发展趋势**：深度学习（CNN/Transformer）、3D视觉、多模态融合、自动驾驶、医学影像AI

## 6. 颜色空间详解

### 通俗解释

颜色空间就是"用数字描述颜色的方法"。就像描述一个人可以用（姓名、年龄、身高），描述颜色也可以用不同的"坐标系"。

### 📌 背记要点

| 颜色空间        | 类型    | 核心特点        | 典型应用    | 是否线性      |
| ----------- | ----- | ----------- | ------- | --------- |
| **RGB**     | 加色    | 三原色光混合      | 显示器、摄像头 | ✅ 线性      |
| **CMYK**    | 减色    | 颜料叠加+黑色K    | 打印、印刷   | ✅ 线性      |
| **XYZ**     | CIE标准 | 标准三刺激值，Y=亮度 | 色度学基准   | ✅ 线性      |
| **HSV/HSI** | 极坐标   | H色调S饱和度V明度  | 颜色检测、分割 | ❌ **非线性** |
| **Lab**     | 感知均匀  | 数值等距=视觉等距   | 色彩管理、PS | ❌ 非线性     |
| **YUV**     | 亮度+色度 | 亮度色度分离      | 视频编码传输  | ✅ 线性      |

⚠️ **高频考点**：
- **HSV/HSI是非线性的**（用了极坐标/角度表示）
- HSV三成分是 **HS((色调)、饱和度)、V(明度)**，**不含"对比度"**！
- RGB是加色模型（光叠加），CMYK是减色模型（颜料叠加）

🔤 **口诀**：RGB加色显示，CMYK减色印刷，HSV非线性，YUV视频编码

## 7. 空间分辨率与灰度级

### 通俗解释

**空间分辨率**：图像有多少个像素。比如256×256的图有65536个像素。像素越多，细节越精细。

**灰度级**：每个像素的亮度分几档。灰度级L越多，颜色过渡越平滑。

### 📌 背记要点

- **存储公式**：总bit数 = 宽 × 高 × 每像素bit数
- **每像素bit数** = log₂(L)，L是灰度级数
  - L=8 → 3bit；L=16 → 4bit；L=256 → 8bit
- **换算**：1 Byte = 8 bit，1 KB = 1024 Byte

✅ **真题例**：256×256图，8灰度级(3bit) → 256×256×3 = 196608 bit ≈ **196 Kb**

---

## 8. 点运算 vs 局部运算

### 通俗解释

**点运算**：只看当前这一个像素的值来决定输出，跟周围没关系。就像修图时调亮度——每个像素独立加一个值。

**局部运算**：要看当前像素和周围一圈邻域的值来决定输出。就像模糊滤镜——要算周围一圈的平均值。

### 📌 背记要点

| | 点运算 | 局部运算 |
|--|--------|----------|
| 输入 | 单个像素 | 邻域（周围一圈像素） |
| 典型操作 | 亮度调整、对比度拉伸 | 模糊、锐化、边缘检测 |
| 公式 | g = f(x) | g = f(邻域) |

---

## 9. 线性与非线性点运算

### 9.1 线性点运算

**公式**：`g = a × f + b`

| 参数条件 | 效果 | 直觉理解 |
|----------|------|----------|
| a > 1 | 对比度**增大** | 亮的更亮，暗的更暗，灰差拉大 |
| 0 < a < 1 | 对比度**减小** | 亮暗差距缩小，图像发灰 |
| a = 1, b > 0 | **变亮** | 整体每个像素加正值 |
| a = 1, b < 0 | **变暗** | 整体每个像素减去值 |
| a < 0 | **明暗反转** | 暗变亮、亮变暗（底片效果） |

🔤 **口诀**：a管对比度（>1拉大，<1压缩），b管亮度（>0亮，<0暗），a负则底片效果

### 9.2 非线性点运算

| 变换 | 公式 | 效果 | 应用 |
|------|------|------|------|
| **对数变换** | s = c × log(1+r) | 扩展暗区细节，压缩亮区 | 频谱显示 |
| **伽马变换** | s = c × r^γ | γ<1变亮暗区，γ>1变暗亮区 | 显示器伽马校正 |
| **指数变换** | s = c × (e^r - 1) | 扩展亮区，压缩暗区 | 与对数相反 |

✅ **伽马直觉**：γ<1时曲线在对角线上方→把暗像素拉亮；γ>1时曲线在对角线下方→把亮像素压暗

## 10. 直方图均衡化

### 通俗解释

直方图就像一张"统计表"，横轴是亮度值（0最黑，L-1最白），纵轴是这个亮度值有多少个像素。

如果直方图都挤在一起（比如集中在中间灰），图像看起来就灰蒙蒙的、对比度低。**直方图均衡化**的目的就是把挤在一起的灰度拉伸开来，让图像更清晰。

### 📌 背记要点

**算法4步（必背）**：

1. **统计概率**：p(k) = 灰度k的像素数 / 总像素数
2. **求累积分布（CDF）**：s(k) = p(0) + p(1) + ... + p(k)
3. **映射到新灰度**：新灰度 = **round( s(k) × (L-1) )**
4. **合并输出**：多个原灰度映射到同一新灰度时，概率相加

**特点**：
- 是**非线性灰度变换**
- 对灰度分布**集中/不均匀**的图像效果**最明显**
- 对灰度分布**已经均匀**的图像，效果**不明显**

### 真题完整示范

8灰度级(0~7)，p = [0.174, 0.088, 0.086, 0.080, 0.068, 0.058, 0.062, 0.384]

| 原灰度k | p(k) | CDF s(k) | s×7 | round取整=新灰度 |
|---------|------|----------|-----|-----------------|
| 0 | 0.174 | 0.174 | 1.218 | **1** |
| 1 | 0.088 | 0.262 | 1.834 | **2** |
| 2 | 0.086 | 0.348 | 2.436 | **2** |
| 3 | 0.080 | 0.428 | 2.996 | **3** |
| 4 | 0.068 | 0.496 | 3.472 | **3** |
| 5 | 0.058 | 0.554 | 3.878 | **4** |
| 6 | 0.062 | 0.616 | 4.312 | **4** |
| 7 | 0.384 | 1.000 | 7.000 | **7** |

✅ **答案**：0→1, 1→2, 2→2, 3→3, 4→3, 5→4, 6→4, 7→7

🔤 **口诀**：累加 → 乘(L-1) → 四舍五入

⚠️ **易错点**：
- L=8时乘**7**，L=16时乘**15**，通用乘 **(L-1)**
- 取整用**四舍五入**

## 11. 形态学运算：腐蚀与膨胀

### 通俗解释

形态学运算就像用一个小模具（结构元素）在图像上滑动，根据模具盖住的区域来决定中心像素是留还是去。

- **腐蚀**：像被酸腐蚀了一样，边界**向内收缩**
- **膨胀**：像目标在"膨胀"生长，边界**向外扩张**

### 📌 背记要点

| 操作 | 规则 | 效果 | 口诀 |
|------|------|------|------|
| **腐蚀** | 结构元素**完全包含**在目标内才保留 | 目标**变小/收缩** | 取**min** |
| **膨胀** | 结构元素与目标有**任何交集**就保留 | 目标**变大/扩张** | 取**max** |
| **开运算** | 先腐蚀后膨胀 | 去除小噪声点 | 去噪 |
| **闭运算** | 先膨胀后腐蚀 | 填补小孔洞 | 填洞 |

🔤 **口诀**：腐蚀变小取min，膨胀变大取max；开运算去噪，闭运算填洞

## 12. 卷积运算基础

### 通俗解释

卷积就是拿一个小矩阵（卷积核/滤波器）在图像上滑动，每个位置上做"对应元素相乘再求和"，得到输出图像在该位置的值。

**卷积与相关运算的区别**：卷积要先把核**翻转180°**再乘加，相关运算不翻转。但如果核是中心对称的（如Sobel、高斯核），翻转后不变，所以实际可以省略翻转。

### 📌 背记要点

**输出大小公式（valid卷积，不填充）**：
```
输出大小 = (H - k + 1) × (W - k + 1)
```
H=输入高，W=输入宽，k=核边长

**乘法次数**：
```
总乘法次数 = (H - k + 1) × (W - k + 1) × k²
```

✅ **真题例**：10×15图 + 7×7核 → (10-7+1)×(15-7+1)×7² = 4×9×49 = **1764次**

**卷积核大小经验公式**：
```
k ≈ 2 × 3σ + 1
```
覆盖高斯分布99.7%的面积。σ越大→核越大→平滑越强。

**可分离性**：某些2D核可分解为两个1D核的外积，复杂度从 O(k²) 降为 O(2k)

**卷积性质**：
| 性质 | 含义 |
|------|------|
| 反褶 | 核需翻转180° |
| 交换律 | f * g = g * f |
| 结合律 | (f*g)*h = f*(g*h) |
| 可分离性 | 2D→两个1D，O(k²)→O(2k) |
| 分配律 | f*(g+h) = f*g + f*h |

## 13. 空间域滤波器

### 📌 背记要点

#### 均值滤波器
- 3×3模板：每个位置都是 **1/9**
- 类型：**线性**滤波器、**低通**滤波器
- 特点：对**高斯噪声**效果好；会**模糊边缘**

#### 中值滤波器
- 操作：窗口内所有像素**排序取中值**（3×3取第5个）
- 类型：**非线性**滤波器（**不能写成固定模板！**）
- 特点：对**椒盐噪声**效果极好；**保留边缘**；对高斯噪声不如均值

#### 高斯滤波器
- 以高斯函数为权重的加权平均，中心权重最大
- 类型：**线性**滤波器、**低通**滤波器
- 特点：**各向同性**（各方向均匀平滑）；σ越大平滑越强

**噪声适应性对比（必背）**：

| 噪声类型 | 特点 | 最佳滤波器 | 原因 |
|----------|------|-----------|------|
| **高斯噪声** | 每点都被污染，幅度正态分布(均值=0) | **均值滤波** | 均值=0，取平均可消除 |
| **椒盐噪声** | 只有部分像素被污染（极黑或极白） | **中值滤波** | 极端值排序后不在中值位置 |

🔤 **口诀**：均值治高斯，中值治椒盐

---

## 14. 傅里叶变换与频域滤波

### 通俗解释

傅里叶变换的核心思想：任何信号都可以分解为一系列不同频率的正弦/余弦波的叠加。

在图像中：
- **低频** = 灰度缓慢变化的区域（平坦区域、大面积色块）
- **高频** = 灰度剧烈变化的区域（边缘、噪声、纹理）

中心化后的频谱图：**中心是低频，越靠近边缘频率越高**。

### 📌 背记要点

**卷积定理**：空域卷积 = 频域相乘（反之亦然）`f*g ↔ F·G`

**频域滤波步骤**：
1. 对图像做2D傅里叶变换 → F(u,v)
2. 选滤波器H(u,v)，相乘：G(u,v) = F(u,v) × H(u,v)
3. 对G(u,v)做傅里叶逆变换 → 滤波后图像

| 滤波类型 | 操作 | 效果 | 应用 |
|----------|------|------|------|
| **低通** | 让低频通过，抑制高频 | 平滑/去噪（模糊边缘） | 去噪 |
| **高通** | 让高频通过，抑制低频 | 锐化/增强边缘 | 边缘检测 |

**三种频域滤波器**：

| 滤波器 | 特点 | 振铃效应 |
|--------|------|----------|
| **理想** | 截止频率处硬截断（0→1无过渡） | **有严重振铃** |
| **高斯** | 平滑过渡（高斯曲线形状） | **无振铃** |
| **Butterworth** | 可调过渡（阶数n越大越接近理想） | 低阶无，高阶有 |

✅ **振铃效应**：理想滤波器截止频率的突变会在图像中产生波纹状伪影

⚠️ **二维傅里叶变换具有可分离性**（可拆成两次一维变换）——选择题常考"不具有可分离性"是错的

## 15. 边缘检测算子

### 通俗解释

边缘就是图像中灰度值**剧烈变化**的地方——物体的轮廓、纹理变化等。检测边缘的思路就是**求微分**：灰度变化大的地方，微分值就大。

- **一阶微分**：看灰度变化率（梯度），梯度大→有边缘
- **二阶微分**：看灰度变化率的变化率，**过零点**→有边缘

### 📌 背记要点

#### 一阶微分算子

| 算子 | 核大小 | 特点 | 模板关键 |
|------|--------|------|----------|
| **Roberts** | 2×2 | 最简单，对噪声**最敏感** | 对角差分 |
| **Prewitt** | 3×3 | 无权值，各方向同等权重 | 权重全是1 |
| **Sobel** | 3×3 | 中心加权**2倍**，**抗噪最好** | 权重1-2-1 |

**Sobel模板（必背）**：
```
Gx:              Gy:
-1  0  +1        -1  -2  -1
-2  0  +2         0   0   0
-1  0  +1        +1  +2  +1
```

#### 二阶微分算子

| 算子 | 特点 |
|------|------|
| **Laplacian** | 3×3，各向同性，对噪声**非常敏感** |
| **LOG** | 先高斯平滑再Laplacian，既保留二阶优点又解决噪声问题 |

**Laplacian 4邻域模板**：
```
 0  -1   0
-1   4  -1
 0  -1   0
```

**抗噪排序（必背）**：**Sobel > Prewitt > Roberts > Laplacian**

**一阶 vs 二阶对比**：

| 对比项 | 一阶（Roberts/Prewitt/Sobel） | 二阶（Laplacian/LOG） |
|--------|------------------------------|----------------------|
| 检测原理 | 梯度幅值（灰度变化率） | 过零点（变化率的变化率） |
| 边缘粗细 | 较粗（多像素宽） | 较细（单像素宽） |
| 噪声敏感度 | 较低 | **更高** |
| 定位精度 | 一般 | 更好 |

⚠️ **判断题考点**：「二阶比一阶对噪声敏感度低」→ **错！二阶更敏感！**

## 16. Canny边缘检测算法

### 📌 背记要点

**Canny四步（必背）**：

1. **高斯平滑**：用一阶高斯偏导核卷积图像（同时平滑+求导）
2. **计算梯度幅值和方向**：G = √(Gx²+Gy²)，θ = arctan(Gy/Gx)
3. **非极大值抑制**：沿梯度方向检查是否是局部最大值，非最大值→抑制为0（使边缘变单像素宽）
4. **双阈值检测与边缘连接**：
   - 梯度 > 高阈值Th → **强边缘**（确定是边缘）
   - 梯度 < 低阈值Tl → **舍弃**
   - Tl < 梯度 < Th → **弱边缘**，只有与强边缘相连才保留（8邻域连接）

🔤 **口诀**：高斯平滑 → 求梯度 → 非极大值抑制 → 双阈值连接

⚠️ **Canny不涉及傅里叶变换！** 选择题常把"傅里叶变换"作为干扰项。

## 17. Hough变换与图像分割

### Hough变换

**核心思想**：图像空间中的一个点 → 参数空间中的一条正弦曲线。多个点共线 → 它们在参数空间中的曲线交于同一点。

- 直线检测公式：`ρ = x·cosθ + y·sinθ`
- 在参数空间建立累加器，每个边缘点投票，票数最多的(ρ,θ)对应检测到的直线
- 适用范围：直线、圆、抛物线、椭圆等**可参数化**的形状

### 图像分割方法

| 方法类别 | 代表方法 | 基本原理 | 特点 |
|----------|----------|----------|------|
| **阈值分割** | 全局阈值、Otsu | 选灰度阈值T，>T为前景 | 最简单快速 |
| **边界分割** | 边缘检测+连接 | 检测边缘再连接成封闭区域 | 依赖边缘质量 |
| **区域分割** | 区域生长、分裂合并 | 从种子点出发合并相似像素 | 能产生封闭区域 |

## 18. 聚类方法

### 📌 背记要点

| 方法 | 是否需要预设K | 优点 | 缺点 |
|------|-------------|------|------|
| **k-means** | ✅ 需要 | 简单快速 | 对初值敏感，只能找球形簇 |
| **Mean Shift** | ❌ 不需要 | 任意形状簇 | 计算量大 |
| **层次聚类** | ❌ 不需要 | 树状图直观 | O(n²logn) 复杂度高 |

**k-means步骤**：
1. 随机选k个初始中心
2. 每个样本分配到最近的中心
3. 重新计算每个簇的中心（均值）
4. 重复2-3直到收敛

⚠️ **k-NN是分类器，不是聚类方法！** k-NN需要标签（监督学习），聚类不需要标签（无监督学习）。选择题常将k-NN混入聚类选项。

## 19. 关键点（角点）检测

### 📌 背记要点

**角点定义**：景物轮廓线上曲率的**局部极大点**（注意是**极大**不是极小！），即边缘的拐角处。

#### Moravec角点检测
- 原理：窗口内向8个方向移动1像素，计算灰度差平方和，取所有方向的**最小值**
- 缺点：只有8个方向，不是各向同性，对噪声敏感

#### Harris角点检测
- 构建自相关矩阵M（二阶矩矩阵）
- **响应函数**：`R = det(M) - k × trace(M)²`，k通常取0.04~0.06
- λ₁、λ₂都大 → **角点**；一大一小 → 边缘；都小 → 平坦区域

## 20. SIFT与SURF特征

### 📌 背记要点

| 特征 | 核心特点 | 描述子维度 | 速度 |
|------|----------|-----------|------|
| **SIFT** | 尺度不变+旋转不变 | **128维** | 较慢 |
| **SURF** | SIFT加速版 | 64维 | 快数倍 |

**SIFT四步**：
1. 构建**高斯差分金字塔(DoG)** → 搜索极值点
2. 亚像素级**精确定位**，剔除低对比度点和边缘响应点
3. 计算邻域梯度方向直方图 → **方向赋值**（实现旋转不变性）
4. 16×16区域分成4×4子区域，每子区域8方向 → **4×4×8=128维描述子**

**SURF改进**：用**积分图**和**盒子滤波**替代SIFT中的高斯差分，速度快数倍

**斑点检测**：用不同尺度的LoG滤波器卷积，当σ与斑点大小匹配时LoG响应最大。关键关系：**斑点半径 r = σ√2**

## 21. 目标检测：Viola-Jones框架

### 📌 背记要点

**Adaboost + Haar + 积分图 + 图像金字塔 → Viola-Jones人脸检测**

| 组件 | 作用 |
|------|------|
| **Haar特征** | 矩形特征（白区像素和 - 黑区像素和），检测边缘/线条/中心 |
| **积分图** | 快速计算任意矩形区域像素和，O(1)复杂度（只需4次查表加减） |
| **图像金字塔** | 图像逐级缩小，等效于不同大小窗口检测 → **尺度不变** |
| **Adaboost** | 多个弱分类器加权组合成强分类器，错分样本权重增大 |

## 22. 图像分类：PCA与LDA

### PCA（主成分分析）

**目的**：找到方差最大的投影方向（信息量最大），实现降维。

**算法5步（必背）**：
1. **数据中心化**：每个样本减去均值μ
2. **计算协方差矩阵**：C = (1/n) × Xᵀ × X
3. **特征值分解**：求C的特征值λ和特征向量w
4. **排序选主成分**：特征值从大到小排序，取前k个对应特征向量组成投影矩阵W
5. **降维**：Y = Wᵀ × X

### LDA（线性判别分析）

**目的**：使类间距离最大化、类内距离最小化

### PCA vs LDA 对比（必背）

| 对比项 | PCA | LDA |
|--------|-----|-----|
| 监督方式 | **无监督**（不需要标签） | **有监督**（需要标签） |
| 优化目标 | 最大化**方差** | 最大化**类间分离度** |
| 使用类别信息 | **否** | **是** |
| 降维上限 | 可降到任意维度 | 最多降到 **(C-1)维** |
| 共同点 | 都是**线性变换** | 都是**线性变换** |

🔤 **口诀**：PCA无监督看方差，LDA有监督看类别

⚠️ **利用类别判别信息是LDA的特点，不是PCA的！**（选择题常考）

### CNN（卷积神经网络）

| 特点 | 说明 |
|------|------|
| 卷积核 | 通过训练**学习得到**（不是人工设计） |
| 多个核 | 每个核学习一种特征 |
| 非线性 | 多层卷积+激活函数(ReLU)→整体非线性 |
| 训练数据 | 需要**带标签**的训练数据集 |

### 分类评估指标

| 指标 | 含义 |
|------|------|
| 混淆矩阵 | 预测vs真实的交叉表 |
| 准确率 | 预测正确 / 总数 |
| 精确率 | 预测为正中真正为正的比例 |
| 召回率 | 真正为正中被正确预测的比例 |
| ROC曲线 | FPR-TPR曲线，AUC越接近1越好 |

⚠️ **均方误差(MSE)是回归指标，不是分类指标！**

## 计算题①：均值滤波 & 中值滤波（10分）

### 题目：5×5的16级灰度图

### 第1步：写出滤波器

**3×3均值滤波器**：
```

1/9  1/9  1/9
1/9  1/9  1/9
```

**3×3中值滤波器**：取窗口内9个数排序后的中值（第5个），无固定模板。

### 第2步：滤波计算（只算内部3×3）

**均值滤波**（每个窗口9个数求和÷9）：

以输出(1,1)为例，窗口取原图行0~2、列0~2：
(1+2+2+1+15+1+2+1+2)/9 = 27/9 = **3.0**

完整结果：
```
3.0  3.0  1.9
2.9  3.1  1.8
1.6  1.6  1.7
```

取整后：
```
3  3  2
3  3  2
2  2  2
```

**中值滤波**（每个窗口9个数排序取第5个）：

以输出(1,1)为例，窗口9个数：1,2,2,1,15,1,2,1,2 → 排序：1,1,1,1,2,2,2,2,15 → 中值=**2**

完整结果（全是2）：
```
2  2  2
2  2  2
2  2  2
```

### 第3步：特点与噪声分析

| 滤波器 | 对哪种噪声好 | 原因 |
|--------|-------------|------|
| 均值滤波 | **高斯噪声** | 正态分布均值=0，取平均可消除 |
| 中值滤波 | **椒盐噪声** | 极端值排序后被挤到两端，中值不受影响 |

# Part 十二：8个高频易错点

---

### ⚠️ 易错点1：卷积核翻转
卷积 = 核**翻转180°**后乘加（Sobel对称所以不变，但概念要知道）

### ⚠️ 易错点2：边界填充
看清题目要求：**填0**还是**只算内部**（valid卷积）。填0后图变大，输出也变大

### ⚠️ 易错点3：直方图均衡化是非线性的
即使原始直方图是线性的，CDF也是**非线性**的，所以均衡化是**非线性拉伸**

### ⚠️ 易错点4：中值滤波是非线性的
中值滤波**不能写成固定卷积模板**，因为它不是乘加操作而是排序取中值

### ⚠️ 易错点5：k-NN是分类器不是聚类
k-NN需要标签，是**监督学习**；聚类不需要标签，是**无监督学习**

### ⚠️ 易错点6：角点是曲率极大点
角点是曲率的**局部极大点**，不是极小点！

### ⚠️ 易错点7：SIFT对尺度有不变性
SIFT = **Scale-Invariant** Feature Transform，对尺度变换**有**不变性

### ⚠️ 易错点8：小高斯核平滑弱
核越小(σ越小)平滑能力**越弱**；大核(大σ)才平滑强。公式 k ≈ 2×3σ+1
