# 模拟卷一 参考答案与解析

## 一、选择题答案

1. **答案 A**。解析：存储大小 = 1920 × 1080 × 8 bit = 16,588,800 bit = 2,073,600 Byte ≈ 2MB。注意 1MB = 1,048,576 Byte，所以约 2MB。

2. **答案 B**。解析：BRDF（Bidirectional Reflectance Distribution Function）描述的是给定入射光方向和反射光方向之间，表面反射辐射的分布关系，即入射光方向与反射光方向之间的能量分布关系。

3. **答案 C**。解析：RGB 到 HSV 的变换涉及三角函数和取极值操作，是非线性变换。RGB→XYZ 是线性矩阵变换，RGB→YUV 是线性变换（加权求和），RGB→CMY 也是线性变换（简单相减）。

4. **答案 A**。解析：HSV 中 H = Hue（色调），S = Saturation（饱和度），V = Value（明度）。注意 V 代表明度而非对比度。

5. **答案 B**。解析：在 g = af + b 中，a 是斜率，控制对比度（灰度级的拉开或压缩程度）；b 是偏移量，控制亮度的增减。

6. **答案 C**。解析：成像几何关系描述的是三维世界点到二维图像平面的投影变换关系（如透视投影），由相机内参和外参决定，与场景中光源的明暗无关。

7. **答案 B**。解析：均方误差（MSE）是回归任务的评估指标，衡量预测值与真实值之间的差异。分类任务应使用准确率、精确率、召回率、F1 值等指标。

8. **答案 C**。解析：CNN 中虽然卷积操作本身是线性的，但网络中引入了激活函数（如 ReLU），使得 CNN 具有非线性表达能力。因此说"所有操作都是线性的"是错误的。

9. **答案 A**。解析：valid 卷积输出尺寸 = (12-3+1) × (12-3+1) = 10×10。输出共 100 个元素，每个元素需要 3×3=9 次乘法，总乘法次数 = 100 × 9 = 900。

10. **答案 C**。解析：拉普拉斯滤波器是二阶微分算子，属于高通滤波器，用于图像锐化/边缘增强。均值滤波和高斯低通滤波都是低通滤波器（平滑），中值滤波是非线性平滑滤波。

11. **答案 B**。解析：腐蚀操作使用结构元素在图像上滑动，取局部最小值，效果是缩小前景（白色）区域，消除小的孤立噪声点和细小连接。膨胀则是扩大前景、填补小孔。

12. **答案 C**。解析：k-近邻（k-NN）是一种有监督的分类方法，不是聚类方法。K-Means、层次聚类、DBSCAN 都是无监督的聚类算法。

13. **答案 A**。解析：二维傅里叶变换的可分离性意味着可以先对每行做一维 FFT，再对每列做一维 FFT（或反之），将二维变换分解为两次一维变换，从而大大降低计算复杂度。

14. **答案 C**。解析：一阶微分算子（如 Sobel、Prewitt）通过计算梯度幅值来检测边缘位置；二阶微分算子（如 Laplacian）通过寻找二阶导数的零交叉点来定位边缘。Sobel 是一阶算子，Laplacian 是二阶算子，二阶算子对噪声更敏感。

15. **答案 C**。解析：中值滤波对椒盐噪声等脉冲噪声有很强的抑制能力，抗噪性最好；均值滤波通过平均降低噪声但会模糊边缘；高斯滤波与均值滤波类似但加权不同。一般认为中值 > 均值 ≈ 高斯，但此处按题意选 C。

16. **答案 C**。解析：Canny 边缘检测的步骤为：①高斯平滑去噪 → ②计算梯度幅值和方向 → ③非极大值抑制 → ④双阈值检测和边缘连接。其中不包含傅里叶变换。

17. **答案 C**。解析：角点的数学定义是图像中曲率取极大值的点，即在该点处沿不同方向的灰度变化都很大。Harris 角点检测器通过自相关矩阵的特征值来近似曲率。

18. **答案 C**。解析：PCA 是无监督降维方法，寻找数据方差最大的投影方向；LDA 是有监督降维方法，目标是最大化类间散度与类内散度之比（Fisher 判别准则）。

19. **答案 A**。解析：SIFT 通过构建图像金字塔（多组、多层），在每一层使用不同尺度的高斯核，然后计算 DoG（Difference of Gaussian）来检测多尺度空间中的极值点，从而实现尺度不变性。

20. **答案 B**。解析：直方图集中在低灰度区意味着图像偏暗。均衡化会将灰度级拉伸分布到更宽的范围，使图像亮度增大、对比度增强。

## 二、判断题答案

1. **✗**。理由：图像的空间分辨率是指图像在空间方向上的采样密度（如 1920×1080 像素），而像素的位深度（如 8bit）称为灰度分辨率或亮度分辨率。两者是不同的概念。

2. **✗**。理由：如果图像的直方图已经是近似均匀分布，说明灰度级已经被充分利用，此时进行直方图均衡化几乎没有效果，变化微乎其微。均衡化主要对灰度集中的图像效果明显。

3. **✗**。理由：均值滤波器是对邻域像素取平均值，属于低通滤波器，用于平滑（去噪），会模糊图像。图像锐化需要高通滤波器（如拉普拉斯算子、高提升滤波等）。

4. **✓**。理由：SIFT 通过多尺度空间检测实现尺度不变性，通过主方向分配实现旋转不变性，同时对一定程度的仿射变换和光照变化也具有鲁棒性。

5. **✓**。理由：高斯核的标准差 σ 越小，高斯函数越集中，平滑效果越弱，保留的图像细节越多。σ 越大，平滑越强，图像越模糊。

6. **✗**。理由：Hough 变换是一种参数空间投票方法，可以推广到检测任意可以用参数方程表示的曲线，如圆形（需要 3 个参数：圆心坐标 a、b 和半径 r）、椭圆等。

7. **✓**。理由：微分运算会放大噪声，二阶微分（二阶导数）比一阶微分（一阶导数）对噪声更加敏感，因为噪声经过两次微分后会被进一步放大。实际中常先平滑再做二阶微分（如 LoG 算子）。

8. **✓**。理由：根据卷积定理，空间域中的卷积运算对应频域中的乘法运算（逐频率分量相乘）。即 F{f*g} = F{f} · F{g}，这是傅里叶变换的重要性质之一。

9. **✗**。理由：图像分割的目标是将图像分成具有不同语义意义的区域，可以利用聚类方法作为工具，但图像分割还涉及边缘检测、区域生长、阈值分割等多种方法，且分割结果需要考虑空间连续性等约束。两者不完全等价。

10. **✓**。理由：在尺度空间理论中，归一化的 LoG（拉普拉斯-高斯）算子在特征尺度 σ 处响应最大时，对应斑点的半径为 r = σ√2。这是因为 LoG 算子的零交叉点恰好在 r = σ√2 处。

### 题2：直方图均衡化

**（1）累积分布函数 CDF**

| 灰度级 k | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|----------|------|------|------|------|------|------|------|------|
| p(rₖ) | 0.04 | 0.10 | 0.15 | 0.20 | 0.18 | 0.12 | 0.13 | 0.08 |
| CDF | 0.04 | 0.14 | 0.29 | 0.49 | 0.67 | 0.79 | 0.92 | 1.00 |

计算过程：
- CDF(0) = 0.04
- CDF(1) = 0.04 + 0.10 = 0.14
- CDF(2) = 0.14 + 0.15 = 0.29
- CDF(3) = 0.29 + 0.20 = 0.49
- CDF(4) = 0.49 + 0.18 = 0.67
- CDF(5) = 0.67 + 0.12 = 0.79
- CDF(6) = 0.79 + 0.13 = 0.92
- CDF(7) = 0.92 + 0.08 = 1.00

**（2）均衡化映射**

公式：sₖ = round(CDF(k) × (L-1))，其中 L=8，L-1=7

| 灰度级 k | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|----------|------|------|------|------|------|------|------|------|
| CDF | 0.04 | 0.14 | 0.29 | 0.49 | 0.67 | 0.79 | 0.92 | 1.00 |
| CDF×7 | 0.28 | 0.98 | 2.03 | 3.43 | 4.69 | 5.53 | 6.44 | 7.00 |
| sₖ (取整) | **0** | **1** | **2** | **3** | **5** | **6** | **6** | **7** |

映射关系：
- 0→0, 1→1, 2→2, 3→3, 4→5, 5→6, 6→6, 7→7

**（3）均衡化后各灰度级的概率**

将原灰度级概率按映射关系合并：

| 新灰度级 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|----------|------|------|------|------|------|------|------|------|
| 概率 | 0.04 | 0.10 | 0.15 | 0.20 | 0 | 0.18 | 0.25 | 0.08 |

- s=0: p=0.04（来自 r=0）
- s=1: p=0.10（来自 r=1）
- s=2: p=0.15（来自 r=2）
- s=3: p=0.20（来自 r=3）
- s=4: p=0（无灰度级映射到4）
- s=5: p=0.18（来自 r=4）
- s=6: p=0.12+0.13=0.25（来自 r=5 和 r=6）
- s=7: p=0.08（来自 r=7）

 0   0   0   0   0

```
0   0   0

```
Gx = (-1)×0 + 0×0 + 1×0 + (-2)×0 + 0×10 + 2×10 + (-1)×0 + 0×10 + 1×10

   = 0 + 0 + 0 + 0 + 0 + 0 + 0 + 20 + 10

## 四、综合题答案

### 题1：人脸识别系统设计

**（1）完整处理流程（5步）：**

1. **图像采集**：通过摄像头采集人脸图像或视频流，获取输入图像。

2. **人脸检测**：使用人脸检测算法（如 Haar 特征 + AdaBoost、HOG + SVM、或基于深度学习的 MTCNN / RetinaFace）在图像中定位人脸区域，返回人脸的边界框坐标。

3. **人脸对齐与预处理**：检测人脸关键点（如眼睛、鼻子、嘴巴位置），根据关键点进行几何归一化（仿射变换），使人脸处于统一的位置和姿态；同时进行光照归一化和尺寸归一化。

4. **特征提取**：从归一化后的人脸图像中提取具有区分性的人脸特征向量（人脸描述符）。

5. **特征匹配与识别**：将提取的特征向量与数据库中已注册的人脸特征进行比对，计算相似度（如欧氏距离、余弦相似度），根据阈值判断是否匹配，输出识别结果。

**（2）特征提取方法（至少3种）：**

- **传统方法**：
  - **PCA（特征脸 Eigenfaces）**：对人脸图像进行主成分分析，提取主要特征向量。
  - **LDA（Fisherfaces）**：线性判别分析，最大化类间散度与类内散度之比。
  - **LBP（局部二值模式）**：提取局部纹理特征，对光照变化有较好鲁棒性。
  - **Gabor 特征**：利用 Gabor 滤波器提取不同方向和尺度的纹理特征。

- **深度学习方法**：
  - **CNN（卷积神经网络）**：如 VGGFace、FaceNet、ArcFace 等，通过深度网络自动学习人脸特征。
  - **ResNet**：残差网络，用于提取深层人脸特征。

**（3）分类/匹配方法：**

- **最近邻分类器**：计算测试人脸特征与数据库中所有已知人脸特征的距离，选择最近的。
- **k-近邻（k-NN）**：取最近的 k 个样本投票决定类别。
- **支持向量机（SVM）**：训练分类超平面进行二分类或多分类。
- **余弦相似度匹配**：计算特征向量间的余弦距离进行匹配。
- **Softmax 分类器**：深度学习中常用的多分类输出层。
- **阈值判断**：设定相似度阈值，超过阈值判为同一人，否则判为陌生人。

## 二、判断题答案

1. **✗**。理由：图像质量不仅取决于空间分辨率，还取决于灰度分辨率（位深度）、信噪比、对比度等因素。空间分辨率高但严重失真或噪声很大的图像，质量并不好。

2. **✓**。理由：如果图像所有像素值相同（如全灰度级 128），其直方图是一个冲激函数，CDF 是阶跃函数，均衡化后的映射结果仍然映射到同一个灰度级，图像不会变化。

3. **✗**。理由：均值滤波器是低通滤波器，通过对邻域取平均来平滑图像，会模糊边缘和细节，不能增强边缘。增强边缘需要用高通滤波器或锐化操作。

4. **✓**。理由：SIFT 特征通过多尺度空间检测实现尺度不变性，通过主方向分配实现旋转不变性，特征描述子本身对平移不敏感（相对于关键点位置描述），因此对平移、旋转、尺度变化都具有不变性。

5. **✗**。理由：高斯核的标准差越大，平滑效果确实越强，但高斯核的有效大小（需要覆盖的范围）也越大，需要更多的计算。实际实现中通常截断高斯核到一定大小，但核越大计算量越大。

6. **✓**。理由：检测圆形需要 3 个参数（圆心坐标 a, b 和半径 r），因此 Hough 变换的参数空间是三维的。类似地，检测椭圆需要 5 个参数，参数空间是五维的。

7. **✗**。理由：Laplacian 是二阶微分算子，对噪声的敏感程度高于一阶微分算子 Sobel。因为二阶导数比一阶导数放大噪声更多。实际应用中 LoG（先高斯平滑再 Laplacian）可以缓解这个问题。

8. **✓**。理由：这是卷积定理的核心内容：f(x) * g(x) ↔ F(u)·G(u)，即空域卷积等价于频域逐元素相乘。这是快速卷积和频域滤波的理论基础。

9. **✓**。理由：图像分割的目标正是将图像划分为若干个互不重叠的区域，每个区域内部的像素具有相似的特征（如灰度、颜色、纹理），不同区域之间有明显的差异。

10. **✗**。理由：在尺度空间理论中，归一化 LoG 算子的响应在尺度 σ 处取得最大值时，对应的斑点半径为 r = σ√2，而不是 r = σ。这个关系来源于 LoG 算子的零交叉点位置。

## 四、综合题答案

### 题1：车牌识别系统设计

**（1）主要处理步骤：**

1. **图像采集**：通过道路监控摄像头或停车场摄像头获取包含车辆的原始图像或视频帧。

2. **车牌定位（检测）**：从整幅车辆图像中精确定位车牌区域的位置，提取车牌的边界框。这是整个系统的关键步骤，直接影响后续识别准确率。

3. **车牌预处理**：对定位到的车牌图像进行灰度化、几何校正（倾斜校正/仿射变换）、尺寸归一化、对比度增强等操作，使车牌图像标准化。

4. **字符分割**：将车牌上的每个字符从车牌图像中逐一分割出来。常用方法包括投影法（水平/垂直投影找分割点）、连通域分析、轮廓检测等。

5. **字符识别**：对分割出的每个字符图像，使用模式识别方法进行识别，将其映射为对应的字符（汉字、字母或数字）。

6. **结果输出与校验**：将识别结果组合成完整的车牌号码，可通过语言模型或规则进行后处理校验（如省份简称+字母+数字的格式约束），输出最终结果。

**（2）车牌定位可使用的图像处理技术（至少3种）：**

- **边缘检测 + 形态学处理**：先用 Sobel 或 Canny 检测边缘，再用膨胀/闭运算连接边缘区域，找到车牌候选区域。
- **颜色分割**：利用车牌的固定颜色特征（如蓝底白字、黄底黑字），在 HSV 颜色空间中进行颜色过滤。
- **投影分析**：对图像做水平和垂直方向的灰度投影，找到灰度变化密集的区域（车牌区域纹理丰富）。
- **连通域分析**：对二值化图像进行连通域标记，筛选出符合车牌长宽比和面积特征的区域。
- **基于深度学习的目标检测**：使用 YOLO、SSD 等目标检测网络直接检测车牌位置。

**（3）字符识别的模式识别方法（至少2种）：**

- **模板匹配**：将待识别字符与预存的标准字符模板进行相似度匹配（如归一化互相关），选择最相似的模板对应的字符。
- **支持向量机（SVM）**：提取字符特征（如 HOG 特征），训练 SVM 分类器进行识别。
- **卷积神经网络（CNN）**：使用深度学习方法，将字符图像输入 CNN 进行端到端的分类识别，如 LeNet-5 架构。
- **k-近邻（k-NN）**：提取字符特征后，与训练集中最近的 k 个样本进行投票分类。
- **随机森林 / 决策树**：基于提取的特征训练集成分类器。
