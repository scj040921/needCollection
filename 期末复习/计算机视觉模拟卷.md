## 一、选择题答案与解析

**1. 答案：B**

解析：
- A错误：增大光圈（减小f值）会**减小**景深，使背景更加模糊，而非增大景深。
- **B正确**：根据成像公式1/f = 1/u + 1/v，焦距f越长，在相同物距u下像距v越大，成像越大（即放大倍率越高）。
- C错误：光圈大小不仅影响进光量，还直接影响景深——光圈越大景深越浅。
- D错误：缩小光圈（增大f值）会减少进光量，使图像亮度降低，而非增大。

**2. 答案：C**

解析：
- A错误：这是对杆体细胞的描述，不是锥体细胞。锥体细胞负责明视觉和颜色感知。
- B错误：杆体细胞主要分布在视网膜**周边**区域，中央凹处几乎只有锥体细胞。
- **C正确**：杆体细胞（约1.2亿个）对光的敏感度很高，负责暗环境下的视觉（夜视），但不感知颜色差异。
- D错误：两者分布不均匀——中央凹以锥体细胞为主，周边以杆体细胞为主，功能也不同。

**3. 答案：A**

解析：
- **A正确**：BRDF（Bidirectional Reflectance Distribution Function）的定义为 fr(ωi, ωr) = dLr(ωr) / (Li(ωi)cosθi dωi)，即出射辐射亮度与入射辐照度之比，描述了光线在物体表面的反射特性。
- B错误：BRDF同时依赖于入射方向ωi和出射方向ωr。
- C错误：BRDF的单位是sr⁻¹（每球面度），不是瓦特。
- D错误：BRDF描述的是物体表面材质的反射特性，与光源无关。

**4. 答案：B**

解析：
- A错误：显示器不是成像过程的基本要素。
- **B正确**：成像三要素为：①照明（光源特性）、②场景/物体（反射/透射特性）、③传感器（成像设备/感光元件）。
- C错误：镜头、快门、光圈是相机的组成部分，不是成像三要素。
- D错误：像素、分辨率、灰度级是图像的属性参数，不是成像三要素。

**5. 答案：C**

解析：
- A正确：深度学习（CNN等）在ImageNet分类、COCO检测等任务上取得突破性成果。
- B正确：GPT-4V、LLaVA等视觉-语言多模态大模型是当前热点。
- **C不正确**：计算机视觉仍面临许多未解决的挑战，如开放世界识别、3D理解、鲁棒性等，远未"完全解决"。
- D正确：三维视觉、视频理解、具身智能（embodied AI）等是新兴方向。

**6. 答案：C**

解析：
- A错误：CMYK是**减色模型**，用于印刷；RGB是加色模型，用于显示器。
- B错误：RGB的三个分量与人眼感知的色调、饱和度、亮度**不直接对应**，这是HSI空间的优势。
- **C正确**：HSI（色调Hue、饱和度Saturation、强度Intensity）颜色空间将亮度与色度分离，更符合人眼的感知方式，在图像处理中广泛使用。
- D错误：YUV中Y表示**亮度**，U和V表示**色度**分量，正好说反了。

**7. 答案：B**

解析：
- A错误：512×512×8bit = 512×512×1 byte = 262144 bytes = 256KB。虽然数值上约256KB，但题目问的是存储空间的精确计算，这个说法本身是正确的，但需要看其他选项。
- **B正确**：8比特灰度级意味着每个像素有2⁸ = 256个可能的灰度值（0~255）。这是灰度级的基本定义。
- C错误：空间分辨率（像素数）和灰度级（每像素比特数）是两个独立的参数，之间没有必然的因果关系。
- D错误：图像总存储空间 = 空间分辨率（像素数）× 灰度级（比特/像素），两者直接相关。

（注：A选项的数值计算结果确实是256KB，但B选项是关于灰度级概念的精确定义，是最本质正确的答案。在考试中，A的表述"约为"也是正确的，但B是最直接考查灰度级概念的选项。）

**8. 答案：B**

解析：
- A错误：点操作只考虑**当前像素**本身的灰度值，不涉及邻域信息；考虑邻域的操作是局部操作。
- **B正确**：对数变换 s = c·log(1+r) 的特点是扩展低灰度值区域（暗区域）的动态范围，压缩高灰度值区域（亮区域）的动态范围。
- C错误：γ>1的幂次变换（s = crᵞ）会压缩整体灰度范围，使图像变暗；γ<1时才会扩展暗区域。
- D错误：分段线性变换可以通过设计不同的折线函数，**灵活地突出**任意感兴趣的灰度区间。

**9. 答案：B**

解析：
- 直接进行二维3×3卷积：每个像素需要9次乘法和8次加法，总复杂度为O(M×N×9)。
- 3×3均值滤波核可以分离为一个3×1列向量和一个1×3行向量：先对每行做1×3卷积（每像素3次乘法），再对结果做3×1卷积（每像素3次乘法），总复杂度为O(M×N×6)。
- 因此可分离卷积的复杂度约为O(M×N×6)，即**B正确**。
- 更一般地，对于k×k的可分离核，直接二维卷积复杂度O(MNk²)，可分离卷积复杂度O(2MNk)。

**10. 答案：B**

解析：
- A错误：理想低通滤波器在截止频率处是**陡峭截止**（无过渡），**会产生严重的振铃效应**。
- **B正确**：巴特沃斯低通滤波器在通带和阻带之间有平滑的过渡带，振铃效应比理想滤波器轻得多。
- C错误：高斯低通滤波器的过渡带**最宽**（最平滑），频率选择性最差，但**不产生振铃效应**。
- D错误：高通滤波器用于**锐化**图像、增强边缘，低通滤波器才用于平滑和去噪。

**11. 答案：B**

## 二、判断题答案与解析

**1. ✗**

解析：图像的空间分辨率取决于**单位长度内的像素数**（如DPI，每英寸点数），而不仅仅是像素总数。同样1000×1000像素的图像，打印在不同尺寸上，空间分辨率不同。因此空间分辨率与图像的物理尺寸有关。

**2. ✗**

解析：直方图均衡化确实可以使灰度分布趋向均匀，但"更清晰"并不是必然结果。对于某些图像（如灰度分布本身已较均匀的图像），均衡化可能会过度增强对比度，产生不自然的效果。均衡化的效果取决于原始图像的灰度分布特性。

**3. ✓**

解析：均值滤波器的频率响应是低通特性——它抑制高频分量（噪声和边缘），保留低频分量（平滑区域），因此是低通滤波器。均值滤波只能实现平滑，无法实现锐化（锐化需要增强高频分量，使用的是高通滤波器或微分算子）。

**4. ✗**

解析：SIFT特征具有尺度不变性，但这只意味着它能在不同尺度下检测到**对应的关键点位置**，并生成**相同维数**的描述子。由于图像内容在不同距离下会因尺度变化而有不同的像素排列，SIFT描述子的**具体数值**不会完全相同。描述子只是通过尺度归一化保证了对尺度变化的**鲁棒性**，使匹配时能够正确对应。

**5. ✓**

解析：高斯滤波器的σ参数控制平滑程度。σ越小，高斯核越"尖锐"，有效平滑范围越小，因此平滑效果越弱，保留的图像细节（高频信息）越多。反之σ越大，平滑越强，细节丢失越多。

**6. ✗**

解析：霍夫变换是一种通用的参数形状检测方法，不仅可以检测直线，还可以检测圆（需要3个参数：圆心坐标a,b和半径r）、椭圆等其他可以用参数方程表示的形状。直线检测只是霍夫变换最经典的应用之一。

**7. ✓**

解析：二阶微分算子（如拉普拉斯算子）对灰度变化的响应比一阶微分更强烈——它对边缘产生零交叉点，但同时也对噪声更加敏感。因此在实际应用中，通常先进行高斯平滑再用二阶算子（如LoG），以抑制噪声的影响。

**8. ✓**

解析：这是傅里叶变换的重要性质之一——卷积定理。即：f(x,y)*g(x,y) ↔ F(u,v)·G(u,v)，空域中的卷积对应频域中的乘积。反之，空域中的乘积对应频域中的卷积。这一性质是频域滤波的基础。

**9. ✗**

解析：虽然图像分割可以看作像素聚类的一种形式，但并非所有聚类算法都适合直接用于图像分割。图像分割还需要考虑空间邻域关系（空间连续性、边缘信息等），简单的聚类（如仅按灰度值聚类）可能产生不连续的分割区域。此外，图像分割还包括基于阈值、基于边界、基于区域等多种方法，并非都是聚类。

**10. ✓**

解析：在高斯尺度空间中，尺度归一化的拉普拉斯算子（LoG）的极值点对应于斑点位置。对于二维高斯函数的二阶导数，当特征尺度σ与斑点半径r满足r = σ√2时，LoG响应达到最大值。这是斑点检测的基本原理。

70, 80, 90, 120, 130, 160, 170, 180, 200

中值（第5个值）= **130**

### 计算题3：边缘检测（Sobel算子）

**原始图像矩阵：**

```
  0   0   0   0   0
  0  10  20  30   0
  0  40  80 120   0
  0  70 140 210   0
  0   0   0   0   0
```

**（1）计算像素(2,2)处的梯度Gₓ和Gᵧ：**

像素(2,2)的3×3邻域为：
```
10   20   30
40   80  120
70  140  210
```

**水平梯度Gₓ**（使用Sobel Gₓ算子卷积）：

Gₓ = (-1)×10 + 0×20 + 1×30 + (-2)×40 + 0×80 + 2×120 + (-1)×70 + 0×140 + 1×210

= -10 + 0 + 30 + (-80) + 0 + 240 + (-70) + 0 + 210

= -10 + 30 - 80 + 240 - 70 + 210

= **320**

**垂直梯度Gᵧ**（使用Sobel Gᵧ算子卷积）：

Gᵧ = (-1)×10 + (-2)×20 + (-1)×30 + 0×40 + 0×80 + 0×120 + 1×70 + 2×140 + 1×210

= -10 + (-40) + (-30) + 0 + 0 + 0 + 70 + 280 + 210

= -10 - 40 - 30 + 70 + 280 + 210

= **480**

**（2）计算梯度幅值和方向：**

梯度幅值：
|G| = √(Gₓ² + Gᵧ²) = √(320² + 480²) = √(102400 + 230400) = √332800 ≈ **577**

梯度方向：
θ = arctan(Gᵧ/Gₓ) = arctan(480/320) = arctan(1.5) ≈ **56.3°**

（该方向表示边缘的法线方向，即边缘方向约垂直于此，约为33.7°）

设定阈值T = 100。

由于|G| = 577 > T = 100，因此**该像素是边缘像素**。

## 四、综合题答案与解析

### 人脸识别系统

**（1）人脸识别系统的整体流程：**

人脸识别系统通常包括以下步骤：

① **人脸检测**：从输入图像中定位人脸区域。常用方法包括Viola-Jones（Haar特征+Adaboost）、基于深度学习的目标检测方法（如MTCNN、RetinaFace等）。输出人脸的位置和大小。

② **人脸对齐/预处理**：对检测到的人脸区域进行几何归一化（如根据眼睛位置进行旋转对齐）和灰度归一化（直方图均衡化、光照补偿等），消除姿态、光照等因素的影响。

③ **特征提取**：从归一化的人脸图像中提取具有区分性的特征向量，将高维图像数据映射到低维特征空间。这是人脸识别的核心步骤。

④ **分类/识别**：将提取的特征与数据库中的已知人脸特征进行比较，确定身份。分为人脸验证（1:1判断是否同一人）和人脸辨识（1:N确定是谁）两种模式。

⑤ **输出结果**：输出识别的身份标签或验证结果。

**（2）常用的特征提取方法：**

① **PCA（特征脸/Eigenface）**：将人脸图像视为高维向量，通过PCA找到训练样本协方差矩阵的主成分（特征脸），将人脸投影到低维子空间中。优点是简单高效，缺点是对光照和表情变化敏感。

② **LDA（Fisherface）**：使用线性判别分析，找到使类间散度与类内散度之比最大的投影方向。与PCA不同，LDA利用了类别标签信息，提取的特征更具判别性。

③ **LBP（局部二值模式）**：将每个像素与其邻域像素比较，生成二进制编码，然后统计局部区域的LBP直方图作为特征。LBP对光照变化鲁棒，计算简单。

④ **深度学习特征（如FaceNet/ArcFace）**：使用深度卷积神经网络（如ResNet）学习人脸特征嵌入，通过度量学习（如三元组损失、ArcFace损失）使同一人的特征聚集、不同人的特征分离。

**（3）常用的分类/识别方法：**

① **最近邻分类器（1-NN）**：将待识别人脸的特征向量与数据库中所有已知人脸的特征向量计算距离（如欧氏距离、余弦相似度），选择距离最近的作为识别结果。简单直观，适合小规模数据库。

② **支持向量机（SVM）**：在特征空间中找到最优超平面，最大化不同类别之间的间隔。对于二分类人脸验证问题效果好，可通过核函数处理非线性问题。

③ **神经网络分类器**：使用全连接层或softmax层对提取的深度特征进行分类。在深度学习方法中，特征提取和分类通常集成在同一个网络中端到端训练。

## 二、判断题答案与解析

**1. ✓**

解析：空间分辨率是衡量图像可辨别最小细节能力的指标，通常用DPI（dots per inch）或PPI（pixels per inch）表示，即单位长度内包含的像素数。像素数越多、单位长度越小，空间分辨率越高。

**2. ✓**

解析：直方图均衡化的变换函数s = T(r) = (L-1)×C(r)是由累积分布函数决定的，这是一个**非线性**映射（除非原始分布本身就是均匀分布）。它通过重新分布灰度值来自动增强图像对比度。

**3. ✗**

解析：均值滤波器是一种**低通滤波器**，它抑制高频分量（包括噪声和边缘），使图像变得更平滑（更模糊）。它**不能**增强边缘细节——增强边缘需要使用高通滤波器（如锐化模板、微分算子等）。

**4. ✓**

解析：SIFT算法通过构建图像的高斯尺度空间（使用不同σ的高斯滤波器），并计算相邻尺度的差分（DoG，高斯差分）来近似尺度归一化的LoG算子。在DoG空间中寻找极值点作为关键点，这些关键点同时具有位置和尺度信息，从而实现尺度不变性。

**5. ✓**

解析：高斯滤波器的有效作用范围约为3σ（每侧），因此卷积核的大小通常取为(2×⌊3σ⌋+1)×(2×⌊3σ⌋+1)。σ越大，需要的核尺寸越大，否则截断会导致信息丢失和滤波效果不准确。

**6. ✓**

解析：这是霍夫变换的核心原理。在图像空间中，一条直线y = mx + b上的所有点，映射到参数空间(m,b)中对应一组直线，这些直线在参数空间中交于一点(m₀,b₀)，即对应原直线的参数。这就是霍夫变换检测直线的基本思想。

**7. ✓**

解析：二阶微分算子对灰度变化的响应比一阶微分更剧烈——它对灰度梯度变化产生零交叉，对噪声点也会产生强烈的正负振荡。因此二阶算子对灰度变化更敏感，但也更容易受噪声干扰，在使用前通常需要先进行平滑处理。

**8. ✓**

解析：卷积定理：f(x,y) * g(x,y) ↔ F(u,v)·G(u,v)，即空域卷积等价于频域相乘。这是傅里叶分析中最重要的定理之一，也是频域滤波的理论基础。反之，空域相乘对应频域卷积。

**9. ✗**

解析：图像分割不仅仅是聚类问题。虽然某些分割方法（如K-means图像分割）确实基于聚类思想，但图像分割还包括基于阈值的方法（如Otsu）、基于边界的方法（如边缘检测+边缘连接）、基于区域的方法（如区域生长、分水岭）等。此外，纯聚类方法忽略了像素间的空间邻域关系，直接用于图像分割可能产生空间不连续的分割结果。

**10. ✓**

解析：在高斯尺度空间中，尺度归一化的LoG（拉普拉斯-高斯）算子在尺度σ处对半径为r = σ√2的斑点响应最大。这一关系来源于LoG算子的数学特性，是斑点检测（如SIFT中的DoG近似）的理论基础。

### 计算题1：均值滤波与中值滤波

**原始图像矩阵：**

```
 50  50  50  50  50
 50  50  50  50  50
 50  50   5  50  50
 50  50  50  50  50
 50  50  50  50  50
```

**（1）3×3均值滤波，像素(2,2)处：**

像素(2,2)的3×3邻域为：
```
50  50  50
50   5  50
50  50  50
```

均值 = (50+50+50+50+5+50+50+50+50) / 9 = 405 / 9 = **45**

**（2）3×3中值滤波，像素(2,2)处：**

将3×3邻域内的9个值排序：
5, 50, 50, 50, 50, 50, 50, 50, 50

中值（第5个值）= **50**

**（3）效果分析：**

- 原始像素(2,2)的值为5，与其8邻域（全为50）相比明显偏低，是一个椒盐噪声暗点。
- **均值滤波**结果为45，虽然将5提升到了45，但与周围像素值50仍有差距（差5），说明噪声**部分残留**，且拉低了周围区域的整体灰度值。
- **中值滤波**结果为50，**完全恢复**了该像素应有的正常灰度值（与周围像素一致），噪声被完全去除。
- **结论**：中值滤波**更有效**。原因：椒盐噪声表现为极值点（极大或极小），中值滤波取排序后的中间值，可以完全排除极值的影响。在9个像素中有8个正常值（50）和1个噪声值（5）的情况下，中值恰好是50（正常值），噪声被完全消除。而均值滤波将噪声值5也纳入了平均计算，导致结果被拉低到45，不能完全消除噪声的影响。

### 计算题3：边缘检测（Sobel算子）

**原始图像矩阵：**

```
  0   0   0   0   0
  0  50  50  50   0
  0  50 100  50   0
  0  50  50  50   0
  0   0   0   0   0
```

**（1）计算像素(2,2)处的梯度：**

像素(2,2)的3×3邻域为：
```
50   50   50
50  100   50
50   50   50
```

**水平梯度Gₓ：**

Gₓ = (-1)×50 + 0×50 + 1×50 + (-2)×50 + 0×100 + 2×50 + (-1)×50 + 0×50 + 1×50

= -50 + 0 + 50 + (-100) + 0 + 100 + (-50) + 0 + 50

= **0**

**垂直梯度Gᵧ：**

Gᵧ = (-1)×50 + (-2)×50 + (-1)×50 + 0×50 + 0×100 + 0×50 + 1×50 + 2×50 + 1×50

= -50 + (-100) + (-50) + 0 + 0 + 0 + 50 + 100 + 50

= **0**

**（2）梯度幅值和方向：**

|G| = √(0² + 0²) = **0**

梯度方向：**未定义**（因为Gₓ和Gᵧ都为0）

**（3）分析：**

像素(2,2)的梯度幅值为0，**不能**正确反映该像素处灰度突变的事实。

原因分析：虽然像素(2,2)的灰度值100明显高于周围像素（50），看起来像是一个灰度突变点，但实际上这是一个**孤立的亮点**（类似于斑点/blob），而非**边缘**。Sobel算子是**方向性**的微分算子——Gₓ检测水平方向的灰度变化，Gᵧ检测垂直方向的灰度变化。在该图像中：

- 水平方向：像素(2,2)左右两侧的像素值相同（都是50），因此水平梯度Gₓ = 0
- 垂直方向：像素(2,2)上下两侧的像素值相同（都是50），因此垂直梯度Gᵧ = 0

这是一个关于**对称性**的问题：像素(2,2)在各个方向上的灰度变化都是对称的（两侧值相同），Sobel算子在计算梯度时正负方向的变化相互抵消，导致梯度为0。

这说明Sobel算子适合检测**边缘**（单方向上的灰度突变），但不适合检测**斑点**或**孤立点**。对于斑点检测，应使用拉普拉斯算子或LoG算子，它们对各向同性的灰度突变更敏感。

## 四、综合题答案与解析

### 车牌识别系统

**（1）整体流程：**

① **图像采集**：通过摄像头获取车辆图像（可能在不同光照、角度、距离条件下拍摄）。

② **车牌定位**：从车辆图像中自动定位车牌区域。车牌通常具有固定的长宽比、颜色特征和纹理特征，可利用这些先验知识进行定位。

③ **车牌矫正**：对倾斜、透视变形的车牌图像进行几何校正（仿射变换或透视变换），使车牌变为标准矩形。

④ **字符分割**：将车牌上的各个字符（汉字、字母、数字）逐一分割出来。可利用垂直投影法、连通域分析等方法。

⑤ **字符识别**：对每个分割出的字符图像进行分类识别，输出字符标签。

⑥ **结果输出**：将识别的字符序列组合为车牌号码，输出最终结果。

**（2）各步骤中使用的图像处理与模式识别方法：**

① **车牌定位**：
- 基于颜色的方法：利用车牌底色（蓝底白字、黄底黑字等）在HSV或其他颜色空间中进行颜色过滤。
- 基于边缘的方法：使用Sobel算子检测水平边缘，利用车牌区域边缘密集的特征进行定位。
- 基于形态学的方法：对二值化后的图像进行膨胀和连通域分析，筛选出符合车牌长宽比的区域。
- 基于深度学习的方法：使用目标检测网络（如YOLO、SSD）直接检测车牌位置。

② **车牌矫正**：使用仿射变换或透视变换，根据车牌边框的四个角点进行几何校正。

③ **字符分割**：
- 垂直投影法：对二值化车牌图像沿列方向求和，字符间的空隙对应投影的谷底位置。
- 连通域分析：标记连通区域，每个连通域对应一个字符。
- 基于轮廓的方法：检测字符轮廓并提取外接矩形。

④ **字符识别**：
- 传统方法：提取字符特征（如HOG特征、网格特征、投影特征等），使用SVM、K-NN等分类器进行分类。
- 深度学习方法：使用CNN（如LeNet、ResNet）对字符图像进行端到端分类，或使用CRNN（卷积循环神经网络）进行序列识别。
- 模板匹配法：将输入字符与标准模板进行匹配（相关性计算），选择匹配度最高的类别。

**（3）可能遇到的困难及应对策略：**

① **光照变化**：不同时间、天气条件下光照差异大。
→ 应对：使用自适应阈值二值化（如Otsu）、光照补偿、颜色空间转换（HSI中只用H和S分量减少亮度影响）。

② **车牌倾斜/变形**：拍摄角度导致车牌透视变形。
→ 应对：使用仿射变换/透视变换进行几何校正；在检测时使用旋转不变特征。

③ **车牌污染/遮挡**：车牌上有泥污、螺丝遮挡部分字符。
→ 应对：使用深度学习方法提高鲁棒性；图像增强预处理（直方图均衡化、去噪）。

④ **字符粘连/断裂**：二值化后字符之间粘连或字符内部断裂。
→ 应对：调整二值化阈值；使用形态学操作（开运算分离粘连，闭运算连接断裂）；使用基于序列的识别方法（CRNN）避免显式分割。

⑤ **相似字符混淆**：如"0"和"D"、"8"和"B"、"1"和"I"等形状相似的字符。
→ 应对：使用更强的特征描述子；增大训练样本中相似字符的数量；使用置信度反馈机制。
