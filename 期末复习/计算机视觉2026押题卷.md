## 一、选择题答案与解析

---

**1. 答案：B**

**解析：**
- A错误：增大光圈（减小f值）会**减小**景深，使背景更加模糊，而非增大景深。
- **B正确**：景深的定义就是在焦点前后能够保持"足够清晰"的范围。在这个范围内的物体在成像平面上看起来是清晰的。
- C错误：焦距越短，景深越**大**（广角镜头景深深）；焦距越长，景深越浅（长焦镜头容易背景虚化）。
- D错误：景深与多个因素有关：光圈大小、焦距、拍摄距离、传感器尺寸等，不只与光圈有关。

---

**2. 答案：C**

**解析：**
- A错误：杆体细胞约1.2亿个，不是锥体细胞。锥体细胞约600万个。
- B错误：杆体细胞主要分布在视网膜**周边**区域，中央凹处几乎只有锥体细胞。
- **C正确**：杆体细胞对光的敏感度很高，负责暗环境下的视觉（夜视），但不感知颜色差异。这是选择题高频考点。
- D错误：锥体细胞分为三种类型（L型对红光敏感、M型对绿光敏感、S型对蓝光敏感），能区分不同波长的光。

---

**3. 答案：C**

**解析：**
- A错误：BRDF描述的是物体表面的反射特性，与材质密切相关，不是光源的发射特性。
- B错误：BRDF同时依赖于入射方向ωi和出射方向ωr，这是"双向"的含义。
- **C正确**：BRDF（双向反射分布函数）的定义为 fr(ωi, ωr) = dLr(ωr) / (Li(ωi)cosθi dωi)，同时依赖入射方向、出射方向和表面材质。
- D错误：BRDF的单位是sr⁻¹（每球面度），不是瓦特。

---

**4. 答案：C**

**解析：**
- A正确：RGB是线性颜色空间，采用加色模型，用于显示器。
- B正确：CMYK是减色模型（Cyan、Magenta、Yellow、Key/Black），用于印刷。
- **C不正确**：HSV颜色空间是**非线性**的，因为色调H用角度（0°~360°）表示，采用了极坐标形式。HSV更符合人眼感知，但不是线性空间。

**9. 答案：C**

**11. 答案：C**

- B错误：高通滤波器让**高频**通过、抑制低频，用于图像**锐化**（增强边缘），不是平滑。

**12. 答案：C**

**13. 答案：C**

**解析：**

- **高斯滤波**：加权平均能减弱噪声影响，但会模糊边缘，效果不如中值滤波。
- **Laplacian算子直接检测**：二阶微分会放大噪声，产生大量虚假边缘，抗噪性最差。

**15. 答案：B**

**16. 答案：B**

### 计算题4：形态学运算——腐蚀与膨胀

**（3）腐蚀和膨胀的物理意义：**

- **腐蚀**：使目标区域的边界向内收缩。如果结构元素不能完全包含在目标区域内，则该位置的像素被移除。效果是目标变小，可以断开细小连接、去除小噪声点。
- **膨胀**：使目标区域的边界向外扩张。只要结构元素与目标区域有任何交集，该位置就变为前景。效果是目标变大，可以填补小孔洞、连接断裂区域。

### 基于PCA的人脸识别系统（10分）

---

**（1）人脸识别完整流程（4分）：**

```
┌──────────────┐
│ 1. 图像采集    │  采集人脸图像（正面、光照均匀）
└──────┬───────┘
       ▼
┌──────────────┐
│ 2. 预处理      │  ① 灰度化（彩色→灰度）
│              │  ② 几何归一化（裁剪、缩放到统一尺寸如64×64）
│              │  ③ 光照归一化（直方图均衡化消除光照差异）
└──────┬───────┘
       ▼
┌──────────────┐
│ 3. 构建训练集  │  将每张人脸图像展开为一维向量
│              │  N张图像组成数据矩阵X（每列一个人脸向量）
└──────┬───────┘
       ▼
┌──────────────┐
│ 4. PCA降维    │  ① 数据中心化（减去均值脸）

│              │  ③ 特征值分解，取前k个特征向量（特征脸）

└──────┬───────┘
       ▼
┌──────────────┐

└──────┬───────┘
       ▼
┌──────────────┐
│ 6. 识别阶段    │  ① 对测试人脸做相同的预处理
│              │  ② 投影到特征脸空间
│              │  ③ 用分类器判断属于哪个人
│              │  ④ 输出识别结果（身份标签）
└──────────────┘
```

**（2）PCA算法的计算步骤（4分）：**

**Step 1：数据准备与中心化**
- 输入：N个人脸图像，每张展开为d维向量（如64×64=4096维）
- 组成数据矩阵 X = [x₁, x₂, ..., xₙ]，大小为 d×N
- 计算均值向量 μ = (1/N)Σxᵢ
- 中心化：每个样本减去均值 φᵢ = xᵢ - μ
- 得到中心化矩阵 Φ = [φ₁, φ₂, ..., φₙ]

**Step 2：计算协方差矩阵**
- 协方差矩阵 C = (1/N) × Φ × Φᵀ
- C的大小为 d×d（如4096×4096），直接计算非常耗时

**Step 3：特征值分解（PCA的关键技巧）**
- 由于d远大于N，直接求C的特征向量计算量太大
- 技巧：先求小矩阵 L = Φᵀ × Φ（大小为N×N），求L的特征值λᵢ和特征向量vᵢ
- 再由 C的特征向量 uᵢ = Φ × vᵢ 得到原空间的特征向量
- 将特征值从大到小排序：λ₁ ≥ λ₂ ≥ ... ≥ λₙ

**Step 4：选择主成分**
- 选取前k个最大特征值对应的特征向量 u₁, u₂, ..., uₖ
- 选择标准：保留的方差比 ≥ 阈值（如95%），即 Σᵢ₌₁ᵏ λᵢ / Σᵢ₌₁ᴺ λᵢ ≥ 0.95
- 这k个特征向量就是"特征脸"（Eigenface）

**Step 5：降维投影**
- 投影矩阵 W = [u₁, u₂, ..., uₖ]，大小为 d×k
- 对任意人脸x，降维后的特征向量 y = Wᵀ × (x - μ)
- y的维度从d降到k（如从4096降到50），完成降维

**得分要点**：
- 步骤完整（5步）：2分
- 每步有具体公式和解释：2分

---

**（3）局限性与改进（2分）：**

**局限性：**

1. **对光照敏感**：PCA基于像素灰度值，光照变化会显著影响识别结果。同一人在不同光照下的差异可能大于不同人之间的差异。
2. **对姿态敏感**：PCA假设人脸是正面且对齐的，姿态变化（侧脸、低头等）会导致识别率下降。
3. **线性假设**：PCA是线性降维方法，无法捕捉人脸的非线性变化（如表情变化、遮挡等）。
4. **需要类别标签用于训练**：虽然PCA本身无监督，但识别阶段需要标签信息。
5. **计算量大**：当图像尺寸大、训练样本多时，协方差矩阵的特征值分解计算量很大。

**改进方向：**

1. **光照预处理**：在PCA之前加入光照归一化步骤（如直方图均衡化、商图像方法）。
2. **使用LDA替代或补充PCA**：LDA利用类别信息，能找到更有判别力的投影方向（Fisherface方法）。
3. **使用核PCA（Kernel PCA）**：通过核函数将数据映射到高维空间，处理非线性问题。
4. **使用深度学习方法**：如FaceNet、ArcFace等基于CNN的人脸识别方法，能自动学习鲁棒的人脸特征。
5. **数据增强**：通过旋转、翻转、加噪声等方式扩充训练集，提高模型的泛化能力。

**得分要点**：
- 列举至少3个局限性：1分
- 提出合理的改进方向：1分
