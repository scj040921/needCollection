## 《人工智能应用导论》
### 相关描述：
《人工智能导论》是计算机相关专业中一门综合性很强的基础课程，主要内容包括人工智能概述、搜索策略、知识表示、逻辑推理方法、非确定性推理方法、机器学习、人工智能的其他应用领域。 本课程的目的与任务是使学生通过本课程的学习，从人工智能的基本概念入手，由浅入深学习人工智能的各种相关知识，掌握人工智能的相关思想。本课程除要求学生掌握人工智能的基础知识和理论，重点要求学生学会分析问题的思想和方法，为更深入地学习和今后的实践打下良好的基础。
## 《大语言模型API应用与开发实战》
### 课程描述
课程旨在帮助学生了解和掌握大语言模型 API 的开发技术和实践方法。本课程将介绍大语言模型的基本概念和原理，并通过实际案例演示如何使用 API 进行开发。课程内容将包括以下几个方面：1、大语言模型基础知识：介绍大语言模型的背景、应用场景和基本原理，帮助学生了解大语言模型的基本概念和工作原理。2、大语言模型 API 的使用：详细介绍大语言模型 API 的功能和使用方法，包括如何进行模型调用、参数设置、结果解析等，帮助学生快速上手使用 API 进行开发。3、实践案例演示：通过实际案例演示，展示大语言模型 API 在不同应用场景下的具体应用方法。案例涵盖对话生成、微调，嵌入等常见任务，通过实操掌握API的使用。4、API 开发技巧和注意事项：介绍开发大语言模型 API 的一些常用技巧和注意事项，包括如何优化代码性能、处理异常情况、保证系统稳定性等。
## 《Python科学计算基础》
### 课程描述
本课程主要由两大部分组成，python基础及科学计算。通过案例的形式刨析pyhon中，函数，类，对象等知识点，以及科学计算三大神器numpy,Pandas,matpltlab，重点介绍项目案例的编程实现，以及如何使用pandas做科学计算，主要涵盖数组运算、数值计算、代数计算、数据处理等方向，最后用matplotlab 做数据可视化的展示,使学生更好的了解。
## 《RAG知识库实战（智能知识引擎）》
### 课程描述
智能知识引擎实战课程是一个针对人工智能领域的实践性课程，旨在帮助学习者掌握构建和应用智能知识引擎的技能。本课程将提供全面的理论知识和实际操作经验，让学习者能够通过使用知识图谱、自然语言处理和机器学习等技术，构建智能化的知识引擎。
