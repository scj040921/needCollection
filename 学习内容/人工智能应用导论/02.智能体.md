# 智能体与环境
- **智能体**（Intelligent Agent）是一个能感知环境和执行行为的实体，这种实体可以是智能软件、智能设备、智能机器人或智能计算机系统等，甚至也可以是人。
- 智能体嵌入环境中，通过传感器感知环境，通过效应器自洽地作用于环境并满足要求。
- 从数学的角度，**智能体函数将感知序列映射到动作**。
- 从计算机科学的角度，**智能体由运行在机器上的应用程序生成**。![[Pasted image 20260414151915.png]]
- 如图智能体通过传感器感知环境，通过执行器反过来作用于环境。
# 智能体的类型
## 反应式智能体
- 一种具备对当时环境的实时反应能力的智能体。
![[Pasted image 20260414152259.png]]
- **反应式智能体**（Reactive Agent）是一种最基础、最高效的智能体类型，其核心思想是 **“感知 → 行动”**（Perceive-Act），**不进行复杂的内部推理或长期规划**，而是像生物的**条件反射**一样，对环境刺激做出**快速、直接的响应**。
- 特点

| 特性    |              说明              |
| ----- | :--------------------------: |
| 无内部状态 |     不保存历史信息，决策仅依赖当前感知输入      |
| 无世界模型 |        不构建对环境的抽象理解或预测        |
| 规则驱动  | 基于定义的“如果...那么...”（IF-THEN）规则 |
| 响应极快  |         因无需计算推理，延迟极低         |
| 实现简单  |       适合资源受限或实时性要求高的场景       |
## 基于模型的智能体
- 一种基于模型的系统，模型包括环境描述和丰富的智能行为的逻辑推理能力
- 该类型智能体的核心思想是“先理解，再行动”
- 与反应式智能体对比

| 对比维度       | 反应式智能体        | 基于模型的智能体          |
| ---------- | ------------- | ----------------- |
| 是否建模世界     | 无内部模型         | 维护内部状态（世界模型）      |
| 是否记忆历史     | 仅依赖当前感知       | 状态随时间更新（有记忆）      |
| 能否处理不确定性   | 易受噪声干扰        | 通过信息融合提升鲁棒性       |
| 能否实现目标导向行为 | 只能响应刺激        | 明确目标驱动决策          |
| 典型应用       | 扫地机器人避障、简单NPC | 自动驾驶、无人机导航、医疗诊断助手 |
