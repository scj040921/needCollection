# 智能体与环境
- **智能体**（Intelligent Agent）是一个能感知环境和执行行为的实体，这种实体可以是智能软件、智能设备、智能机器人或智能计算机系统等，甚至也可以是人。
- 智能体嵌入环境中，通过传感器感知环境，通过效应器自洽地作用于环境并满足要求。
- 从数学的角度，**智能体函数将感知序列映射到动作**。
- 从计算机科学的角度，**智能体由运行在机器上的应用程序生成**。![[Pasted image 20260414151915.png]]
- 如图智能体通过传感器感知环境，通过执行器反过来作用于环境。
# 智能体的类型
## 反应式智能体
- 一种具备对当时环境的实时反应能力的智能体。
![[Pasted image 20260414152259.png]]
- **反应式智能体**（Reactive Agent）是一种最基础、最高效的智能体类型，其核心思想是 **“感知 → 行动”**（Perceive-Act），**不进行复杂的内部推理或长期规划**，而是像生物的**条件反射**一样，对环境刺激做出**快速、直接的响应**。
- 特点

| 特性    |              说明              |
| ----- | :--------------------------: |
| 无内部状态 |     不保存历史信息，决策仅依赖当前感知输入      |
| 无世界模型 |        不构建对环境的抽象理解或预测        |
| 规则驱动  | 基于定义的“如果...那么...”（IF-THEN）规则 |
| 响应极快  |         因无需计算推理，延迟极低         |
| 实现简单  |       适合资源受限或实时性要求高的场景       |
## 基于模型的智能体
- 一种基于模型的系统，模型包括环境描述和丰富的智能行为的逻辑推理能力
- 该类型智能体的核心思想是“先理解，再行动”
- 与反应式智能体对比

| 对比维度       | 反应式智能体        | 基于模型的智能体          |
| ---------- | ------------- | ----------------- |
| 是否建模世界     | 无内部模型         | 维护内部状态（世界模型）      |
| 是否记忆历史     | 仅依赖当前感知       | 状态随时间更新（有记忆）      |
| 能否处理不确定性   | 易受噪声干扰        | 通过信息融合提升鲁棒性       |
| 能否实现目标导向行为 | 只能响应刺激        | 明确目标驱动决策          |
| 典型应用       | 扫地机器人避障、简单NPC | 自动驾驶、无人机导航、医疗诊断助手 |

- ![[Pasted image 20260414153934.png]]
- **解释上图内容**：
  1. *状态*：
     - 是基于模型的智能体的核心标志。
     - 它表示智能体内部维护的对世界当前状况的估计或表征
     - 与反应式智能体不同：他不是每次只看当前传感器输入，而是持续更新一个内部状态变量。
     - 更新方式：通过信息融合模块，结合传感器输入+先前状态+知识，推断出更准确的当前状态
  2. *信息融合*：
     - 将多源信息整合：
       - 传感器实时数据
       - 内部状态（上一时刻估计）
       - 领域知识
     - 输出：更鲁棒、更完整的世界状态估计（尤其在传感器噪声大或部分紧缺时至关重要）。
  3.  *知识库*：
  
     - 存储关于世界的先验知识，例如：
       - 物理规律
       - 环境拓扑
       - 常识规则
     - 这些知识用于：
       - 辅助状态推断
       - 支持规则推理
  4. *规则*：
     - 不再是简单的“IF-THEN”，可涉及状态变量和目标导向逻辑
  5. *目标*：
     - 明确的期望状态
     - 决策不再仅响应刺激，而是为实现目标服务——这是从“被动反应”到“主动行为”的关键跃升
  6. *作用决策*：
     - 输入：当前目标+当前状态+规则/知识
     - 输出：一个能最有效缩小目标与现状差距的动作
     - 可能使用简单启发式，也可能调用规划算法
  7. *执行器&传感器&环境*：
     - 与反应式智能体相同：通过传感器感知环境，通过执行器改变环境，形成闭环
## 基于目标的智能体
- **基于目标的智能体**（Goal-Based Agent）是人工智能中一种**以明确目标为导向进行决策**的智能体类型。它在“基于模型的智能体”基础上更进一步：**不仅理解当前状态，还知道“要去哪里”，并主动规划路径实现目标**。
- **核心思想**：为达成目标而行动。
  **反应式智能体**：“看到障碍就转弯”  
  **基于模型的智能体**：“我知道我在哪，周围有什么”  
  **基于目标的智能体**：“我要去A点，现在在B点，该怎么走？”
- 关键特征

| 特性        | 说明                       |
| --------- | ------------------------ |
| 显式目标      | 智能体被赋予一个清晰的目标状态          |
| 状态+目标联合决策 | 决策不仅看当前状态，更看当前状态与目标之间的差距 |
| 支持规划      | 能生成动作序列来逐步逼近目标           |
| 处理多可能性    | 当多个动作都可行时，选择最可能达成目标的那个   |
## 基于效用的智能体
- **基于效用的智能体**（Utility-Based Agent）是人工智能中一种**能对多个可能行动进行“满意度”评估，并选择最优方案**的高级智能体。它在“基于目标的智能体”基础上更进一步：**不仅关心“是否达成目标”，更关心“以多好的方式达成目标”**。
- **核心思想**：追求最大满意度，而非仅仅完成任务。
- 通过一个**效用函数**量化每个状态或行动的好坏程度，并选择期望效用最大化的行动
- 关键特征
  

| 特性     | 说明                             |
| ------ | ------------------------------ |
| 效用函数   | 将世界状态映射为一个数值（效用值），代表该状态的“理想程度” |
| 处理冲突目标 | 能在多个目标间权衡                      |
| 考虑不确定性 | 在随机环境中，选择期望效用最高的行动             |
| 支持精细决策 | 当多个行动都能达成目标时，选最好的那个            |
# 智能体的基本特性
## 自主性：
又称自治性，能够在没有人或者别的智能体的干预下，主动地自发地控制自身的行为和内部状态，并且还有自己的目标或意图。
## 反应性：
智能体能对外界的刺激作出反应。对于外界环境的改变，智能体能主动采取行动。
## 社会性：
智能体具有与其它智能体交互或合作的能力。不同的智能体可根据各自的意图与其它智能体进行交互，以达到解决问题的目的。
## 进化性：
智能体能累积或学习经验和知识，并修改自己的行为以适应环境。
