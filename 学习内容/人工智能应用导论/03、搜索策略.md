# 搜索的基本概念
##  1. 采用某种策略，在所规定的约束条件下，从初始实时到目标状态构造一条代价较小的推理路径，使问题得到解决的计算过程称为搜索
# 搜索策略
## 搜索分类
- 盲目搜索就是无“**向导**”的搜索。树式盲目搜索就是穷举式搜索，即从初始节点出发，沿着连接边逐一考察各个节点，或者反向进行。
- 按搜索范围的扩展顺序的不同，搜索又可分为**广度优先搜索**（BFS）和**深度优先搜索**（DFS)
### BFS(广度优先搜索)
#### 基本概念
**广度优先搜索**（Breadth-First Search，简称BFS）是一种用于遍历或搜索树或图的经典算法。它从起始节点开始，**按层次遍历**所有节点，确保先访问所有相邻节点后再深入下一层。
#### 核心思想
1. **水波扩散原理**：
   像水波向外扩散一样，BFS总是先探索距离起点最近的的节点，然后逐步向外扩展。
2. **层次遍历**：
   - 从起点开始，先访问所有直接相邻的节点
   - 再访问这些相邻节点的相邻节点
   - 以此类推，直到遍历完所有可达节点
#### 算法流程
1. 初始化：将起始节点放入队列，标记为已访问
2. 循环执行：
   - 从队列头部取出一个节点
   - 检查该节点是否为目标节点
   - 如果是，返回结果
   - 如果不是，将该节点所有未访问的相邻节点加入队列尾部
   - 标记这些节点为已访问
3. 终止条件：队列为空或者找到目标节点
#### 关键特点

| 特点     | 说明                       |
| ------ | ------------------------ |
| 最短路径保证 | 在无权图中，第一次到达目标节点的路径一定是最短的 |
| 系统性    | 彻底搜索整张图，不依赖经验法则          |
| 避免重复   | 通过标记已访问节点防止循环            |
| 空间复杂度  | O(V),V为节点个数              |
| 时间复杂度  | O(V+E)，V为节点数，E为边数        |
#### 缺点：
盲目性较大，当目标节点距离起始节点较远时将会产生许多无用节点，搜索效率低。
### DFS（深度优先搜索）
#### 基本概念：
**深度优先搜索**（Depth-First Search，简称DFS）是一种用于遍历或搜索树或图的经典算法。它从起始节点开始，**沿着一条路径尽可能深入**，直到无法继续为止，然后回溯到上一个节点，继续探索其他路径。
#### 核心思想：
1. **走迷宫原理**：像一个人在迷宫中探索，总是选择一条路走到黑，直到遇到死路，才返回上一个岔路口，选择另一条路继续前进。
2. **深度优先**：
   - 从起点开始，选择一个未访问的相邻节点
   - 递归地深入该节点，继续选择未访问的相邻节点
   - 当当前节点没有未访问的相邻节点时，回溯到上一个节点
   - 重复这个过程，直到遍历完所有可达节点
#### 算法流程：
1. 初始化：选择起始节点标记为已访问
2. 递归执行：
   - 访问当前节点
   - 选择一个未访问的相邻节点
   - 递归地对该节点执行DFS
   - 如果没有未访问地相邻节点，回溯
3. 终止条件：所有节点都被访问或找到目标节点
#### 关键特点：

| 特点      | 说明                |
| ------- | ----------------- |
| 探索深度    | 有限探索一条路径到最深处      |
| 回溯机制    | 当无法继续深入时返回上一个节点   |
| 空间复杂度   | O(h),h为树的高度       |
| 时间复杂度   | O(V+E),V为节点数，E为边数 |
| 不保证最短路径 | ~                 |
#### 与BFS对比

| 特性    | DFS          | BFS          |
| ----- | ------------ | ------------ |
| 数据结构  | 栈            | 队列           |
| 遍历方式  | 深度优先         | 层次遍历         |
| 最短路径  | 不保证          | 保证最短         |
| 空间复杂度 | 低            | 高            |
| 使用场景  | 路径存在性问题、组合问题 | 最短路径问题、连通性问题 |
| 实现难度  | 简单           | 简单           |
### 启发式搜索
#### 启发式搜索基本概念：
采用问题自身的特性信息，以指导搜索朝着最有希望的方向前进，针对性强，效率高
