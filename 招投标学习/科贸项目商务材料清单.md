# 资格性审查部分
1. 资格文件声明函
2. 有效营业执照（或事业法人登记证或身份证等相关证明）副本复印件 
3. 提供资格文件声明函或提供投标截止日前12个月内任意1个月依法缴纳税收和社会保障资金的相关材料。如依法免税或不需要缴纳社会保障资金的，提供相应的证明材料。
4. 提供资格文件声明函或提供2024年度或2025年度或2026年度至今任意1个月的财务状况报告或基本开户行出具的资信证明。
5. 提供书面承诺声明函或具有履行合同所必需的设备和专业技术能力的其他证明材料。
6. 参加采购活动前3年内，在经营活动中没有重大违法记录：提供书面声明。
# 符合性审查部分
1. 投标函
2. 法定代表人（负责人）证明书、法定代表人（负责人）授权委托书
3. 投标（响应）报价出现异常低价的，对投标（响应）价格作出解释。书面说明、证明材料应包括项目具体成本测算等与报价合理性相关的说明和材料。
# 商务评审
1. 2023年以来完成的系统集成建设类项目或计算机相关购置合同关键页复印件，以合同签订日期为准
2. 项目经理的信息系统项目管理师证书、系统集成项目管理工程师证书的证书复印件，提供上述人员自2026年1月（含1月）来任意1个月投标人（含分支机构）为其缴纳的社保证明复印件（代缴个税税单，或参加社会保险（须至少体现养老保险）的《投保单》等证明均可）
3. 项目团队成员的计算机技术与软件专业技术资格证书复印件，相关工作经历可提供个人简历或同类项目工作情况等证明文件，提供上述人员自2026年1月（含1月）标人（含分支机构）为其缴纳的社保证明复印件（代缴个税税单，或参加社会保险（须至少体现养老保险）的《投保单》等证明均可）。
4. 质量管理体系认证、环境管理体系认证、职业健康安全管理体系认证，以上证书范围须包含计算机相关或销售相关，须提供以上认证的有效认证证书复印件加盖公章，提供认证证书扫描件并须同时提供在全国认证认可信息公共服务平台（http://cx.cnca.cn/）对体系证书的信息查询截图作为评审依据。（证书已暂停、失效或撤销的不得分）
